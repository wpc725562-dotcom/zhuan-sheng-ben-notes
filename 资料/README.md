## 使用

- **Obsidian**：直接打开 `历年真题/公共英语/YYYY` 刷题；需要原卷时来本目录开 PDF。  
- **网站**：只发布 Markdown 笔记，不托管 PDF。  
- 重新生成英语笔记：

```bash
python 资料/_gen_english_notes.py
```
