## 推荐刷法（和真题联动）

```
1. 系统笔记对应章  →  看懂概念
2. 学习辅导该章选择题  →  限时做
3. 学习辅导该章编程题  →  手写 + 上机
4. 真题考点拆分 01–05  →  回炉
5. 2024 全卷  →  掐时间模拟
```

优先顺序（性价比）：

1. **指针 + 数组 + 函数**（计算机分大户）  
2. 选择 / 循环 / 结构体  
3. 文件与综合阅读程序题

## PDF 使用小提示

- 辅导书多为**习题 + 参考答案**，适合「做一章对一章」  
- 若 PDF 是扫描版，本机用阅读器目录跳转最快  
- 答案有争议时：**以教材正文 + 你的编译器实测**为准
