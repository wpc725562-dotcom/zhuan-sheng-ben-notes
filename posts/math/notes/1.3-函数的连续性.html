1.3 函数的连续性

学科系统：高等数学 → 函数与极限 → 函数的连续性 秒杀类比：连续就像一条没有断点的线，而间断点就是线上不小心断开的“缺口”或“跳跃”。

一、 核心知识解构

【是什么】： 函数 y=f(x) 在点 x0 的某一邻域内有定义，如果：

limx→x0f(x)=f(x0)

则称 f(x) 在点 x0 处连续。

另一种等价定义：当自变量变化量 Δx→0 时，函数变化量 Δy→0。

limΔx→0Δy=limΔx→0[f(x0+Δx)−f(x0)]=0

【怎么用】：

  1. 判断某点是否连续 => 检查三要素：左极限、右极限、函数值，三者是否全部相等。
  2. 判断间断点类型 => 只看左、右极限，不看函数值。
    • 左右极限都存在 => 第一类间断点。
      • 左右极限相等 => 可去间断点。
      • 左右极限不相等 => 跳跃间断点。
    • 至少有一个极限不存在 => 第二类间断点。
      • 至少有一个为无穷 => 无穷间断点。
      • 至少有一个为震荡 => 震荡间断点。

【左连续与右连续】：

  • 左连续：f(x0−)=limx→x0−f(x)=f(x0)
  • 右连续：f(x0+)=limx→x0+f(x)=f(x0)
  • 函数在 x0 处连续的充要条件：左连续且右连续，即 f(x0−)=f(x0+)=f(x0)。

二、 统考高频概念与性质汇总

  1. 初等函数的连续性：

    • 一切初等函数在其定义区间内都是连续的。
    • 连续函数经过四则运算、复合后仍然连续。
    • 连续函数的反函数也连续。

  2. 闭区间上连续函数的性质：

    • 有界性与最大值最小值定理：在闭区间上连续的函数一定有界，且能取到最大值和最小值。
    • 零点定理：f(x) 在 [a,b] 上连续，且 f(a)⋅f(b)<0（即一正一负），则至少存在一点 ξ∈(a,b)，使得 f(ξ)=0。
    • 介值定理：f(x) 在 [a,b] 上连续，且端点值不同，则对于介于 f(a) 与 f(b) 之间的任意数 C，至少存在一点 ξ∈(a,b)，使得 f(ξ)=C。

三、 考试红牌警告与秒杀秘籍

🚨 易错雷区：

  1. 判断间断点类型时，千万不要看函数值，只看这点的左右极限。函数值会影响“是否连续”，但不影响“间断点属于哪一类”。
  2. 可去间断点只看左右极限是否相等；跳跃间断点只看左右极限是否不相等；是否属于第一类，只看左右极限是否都存在。
  3. 第二类间断点：只要有一个单侧极限不存在（无穷、震荡等），直接判为第二类。

🔑 秒杀秘籍：

  • 用生活常识理解定理：零点定理就是“一根绳子从水面上一头拉到水面下一头，一定经过水面”；介值定理就是“从3楼（130斤）拉到5楼（180斤），一定经过4楼（150斤）”。
  • 画图比死记硬背更有效。遇到不懂的题目，先画个函数草图，类型一目了然。

四、 闭卷真题挑战 (手脑并用，算完再看解析)

【真题演练】：设函数 f(x)=x2−1x−1，判断 x=1 是 f(x) 的什么类型的间断点？如果改为 f(x)={x,x≠112,x=1，x=1 又是什么类型的间断点？

点击查看答案与解析 【正确答案】：第一个函数：可去间断点；第二个函数：可去间断点。

【核心解析】： 第一问：f(x)=x2−1x−1 在 x=1 处无定义（分母为零），因此是间断点。 计算左右极限：

limx→1x2−1x−1=limx→1(x−1)(x+1)x−1=limx→1(x+1)=2

左极限等于右极限等于 2，但函数值 f(1) 无定义，所以为可去间断点。

第二问：f(x)={x,x≠112,x=1 在 x=1 处有定义，计算左右极限： 当 x→1 时，无论从左或从右，表达式都为 x，故

limx→1f(x)=1

左极限等于右极限等于 1，但 f(1)=12≠1，因此也为可去间断点。

总结：可去间断点只需左右极限相等即可，与函数值无关。
