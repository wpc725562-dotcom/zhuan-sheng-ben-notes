公共英语 · 语法考点精讲

📐 语法考点精讲 ​

专升本英语核心语法 | 时态/语态/从句/虚拟语气/非谓语

一、动词时态（🔥 必考） ​

⏰ 三大核心时态 ​

时态结构用法标志词
一般现在时do/does习惯、真理、状态always, usually, often, every...
一般过去时did过去发生的动作yesterday, last..., ago, in 2020
一般将来时will do将要发生的动作tomorrow, next..., soon, in the future

🔥 完形/语法填空最高频时态 ​

时态结构用法
现在完成时have/has done过去发生对现在有影响 / 持续到现在
过去完成时had done过去的过去（站在过去看更早）
现在进行时am/is/are doing此刻正在发生
过去进行时was/were doing过去某时刻正在发生
现在完成进行时have/has been doing从过去到现在一直持续

真题规律： 80%的时态题在 一般过去时、现在完成时、一般现在时 三者之间选择

📌 现在完成时 vs 一般过去时（必考辨析） ​

维度现在完成时一般过去时
时间焦点现在（关注结果）过去（关注动作本身）
能否用具体时间❌ 不能✅ 可以
例句I have seen the movie.（现在知道内容）I saw the movie last night.

标志词区分：

  • 完成时：already, yet, ever, never, just, recently, so far, since, for
  • 过去时：yesterday, last..., ...ago, in 2020, when I was...

二、被动语态 ​

结构公式 ​

be + done (过去分词)
1

各时态被动语态 ​

时态被动结构
一般现在时am/is/are + done
一般过去时was/were + done
一般将来时will be + done
现在完成时have/has been + done
过去完成时had been + done
情态动词can/must/may + be + done

易错点 ​

  • 主动表被动： need/want/require + doing = need to be done
    • The car needs washing. = The car needs to be washed.
  • 感官动词： look, sound, feel, taste, smell + 形容词（主动表被动）
    • The flower smells good.

三、三大从句（🔥🔥 必考） ​

1️⃣ 定语从句 ​

先行词人物人+物/所有格
主格who/thatwhich/thatthat
宾格who(m)/thatwhich/thatthat
所有格whosewhose/of whichwhose

非限制性定语从句： 不能用 that，用 which / who

The book, which I bought yesterday, is very interesting.

2️⃣ 名词性从句 ​

类型引导词例句
主语从句that / whether / whatWhat he said is true.
宾语从句that / if/whether / wh-I think that he is right.
表语从句that / whether / wh-The question is whether we should go.
同位语从句thatThe news that he won surprised us.

3️⃣ 状语从句 ​

类型引导词
时间when, while, as, before, after, until, as soon as
条件if, unless, as long as, provided that
原因because, since, as, now that
让步although, though, even though, while, no matter...
目的so that, in order that
结果so...that, such...that

四、虚拟语气（🔥 必考） ​

三类条件句对比 ​

类型if从句主句
与现在相反did/werewould/could/might + do
与过去相反had donewould/could/might + have done
与将来相反should do / were to dowould/could/might + do

专升本规律： 80%的虚拟语气题只考前两种（与现在/过去相反）

其他虚拟语气考点 ​

考点结构例句
建议/命令/要求should do（should可省）He suggested that she (should) go.
wish后时态退一格I wish I were a bird.
as if/though时态退一格He talks as if he knew everything.
It's time that...用过去式It's time that we left.
would rather that...用过去式I'd rather you came tomorrow.

动词清单（接 that 虚拟）： suggest, advise, propose, recommend, insist, order, command, require, request, demand

五、非谓语动词（🔥🔥 必考） ​

形式意义用法
to do目的、将来不定式：表示目的/主动/将来
doing主动、进行现在分词/动名词
done被动、完成过去分词表示被动/完成

常考固定搭配 ​

只接 to do 的动词：

want, hope, wish, expect, decide, plan, refuse, promise, afford, agree

只接 doing 的动词：

enjoy, finish, avoid, consider, practice, suggest, mind, keep, imagine

既接 to do 又接 doing 意义不同：

动词to dodoing
remember记得要去做记得做过
forget忘记要去做忘记做过
stop停下来去做停止做
try努力去做尝试做
regret遗憾要去做后悔做过

六、语法填空解题技巧 ​

纯填空（无提示词） ​

空格位置可能填的词
名词前冠词(a/an/the)、限定词、介词
动词前情态动词、助动词
两句之间连词(and, but, or, when, because...)
句中关系词(who, which, that, where...)

有提示词 ​

提示词词性可能变形成
动词时态、语态、非谓语（to do/doing/done）
形容词副词(-ly)、比较级/最高级
名词复数、词性转换（→形容词）

秒杀口诀 ​

时态看标志词，完成时找since/for被动公式be+done，主动表被动要记牢定语从句找先行，虚拟后退一个时非谓语三选一，to do目的doing主动done被动

[VOICE: 语法重点抓三大从句、虚拟语气、非谓语动词，这三个板块占语法考题的七成以上，先把它们吃透！]
