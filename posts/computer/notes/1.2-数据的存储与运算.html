数据的存储与运算 ​

🎯 一句话秒杀：数据类型决定了数据在内存中占多少空间、怎么存、能做什么运算。

一、数据类型全家福 ​

C数据类型
├── 基本类型

│   └── 布尔型：_Bool（C99新增）
├── 构造类型
│   ├── 数组

二、整型数据（重点） ​

整型分类 ​

类型关键字字节数(32位)取值范围
基本整型int4−231∼231−1
短整型short2−32768∼32767
长整型long4−231∼231−1

无符号整型unsigned int40∼232−1

🚨 红牌警告 ​

sizeof(int) 在不同的编译器中可能不同——16位系统是2字节，32/64位系统是4字节 但考试默认：int占4字节，short占2字节

有符号 vs 无符号 ​

c
signed int a = -10;   // 可以存负数（最高位表示符号）
unsigned int b = 10;  // 只能存正数（最高位参与数值）
1
2

存储方式：

  • 正数：原码直接存储
  • 负数：补码存储（原码→取反→加1）
数值原码反码补码
+50000 01010000 01010000 0101
-51000 01011111 10101111 1011

🔑 秒杀秘籍 ​

为什么负数用补码存储？ 因为可以让加减法用同一套电路实现！ 例：3+(−2)=0000 0011+1111 1110=1 0000 0001 → 溢出舍去→ 0000 0001 = 1 ✅

三、字符型数据 ​

c
char ch = 'A';        // 字符用单引号
printf("%c", ch);     // 输出字符 → A
printf("%d", ch);     // 输出ASCII码 → 65

常见ASCII码（必须背！） ​

字符ASCII码(十进制)
'0'48
'A'65
'a'97
'\n'10（换行）
'\0'0（字符串结束标志）

🚨 红牌警告 ​

'0' ≠ 0 ≠ '\0''0' = ASCII 48, 0 = 整型0, '\0' = ASCII 0（字符串结束符） 大小写转换：'A' + 32 = 'a' ✅

四、浮点型数据 ​

类型字节有效位数取值范围
float46~7位±3.4×10−38∼3.4×1038
double815~16位±1.7×10−308∼1.7×10308

🚨 红牌警告 ​

浮点数不能直接用 == 比较相等！ 因为浮点运算有精度误差 ✅ 正确做法：if (fabs(a - b) < 1e-6)

五、运算符与表达式 ​

运算符优先级（必考！） ​

优先级运算符结合性
1（最高）() [] . ->左→右
2! ~ ++ -- +(正) -(负) *(指针) &(取址) sizeof右→左
3* / %左→右
4+ -左→右
5<< >>左→右
6< <= > >=左→右
7== !=左→右
8& (按位与)左→右
9^ (按位异或)左→右
10| (按位或)左→右
11&& (逻辑与)左→右
12|| (逻辑或)左→右
13（最低）= += -= 等赋值右→左

🔑 秒杀秘籍 ​

记口诀：单目右结合，赋值右结合，其他左结合 单目运算符（! ~ ++ -- sizeof）是从右到左结合 赋值运算符（= += 等）也是从右到左

自增自减（高频考点！） ​

c
int a = 5, b;
b = a++;   // 先用a的值：b=5，然后a自增：a=6
b = ++a;   // a先自增：a=6，然后用a的值：b=6
1
2
3
表达式等价于先做什么
a++a; a+=1先用后加
++aa+=1; a先加后用

🚨 红牌警告 ​

考试经典坑： printf("%d %d", i++, ++i); 这种——不要写！ 实际结果取决于编译器实现（未定义行为） 考试如果出了：用"先用后加/先加后用"逐条分析

复合赋值运算符 ​

c
a += 3;    // a = a + 3
a *= b+2;  // a = a * (b+2)  ← 注意：先算右边整体
1
2

🔑 秒杀秘籍 ​

a *= b + 2 ⚡ 等价于 a = a * (b + 2) 不是 a = a * b + 2 复合赋值运算符的右边是一个整体表达式！

5. 判断：浮点数可以用 == 直接比较是否相等

👆 点击展开答案
  1. 4, 1, 4（32位系统，64位也是4）
  2. 65, 97
  3. ⚠️ 这是未定义行为——考试如果硬要你算：
    • 假设从左到右：a++ = 5（a变6）→ ++a = 7（a变7）→ a-- = 7（a变6）
    • 结果：5 + 7 + 7 = 19
    • 但不同编译器可能不同！标准写法不该这样
  4. B. a = a * (b + c) ← 复合赋值右边整体加括号
  5. ❌ 错误——浮点数有精度误差，要用 fabs(a-b) < 1e-6
