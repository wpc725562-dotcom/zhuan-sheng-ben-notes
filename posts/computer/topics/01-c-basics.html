同型 3 题（自测） ​

  1. printf("%d", 5 && 0 || 2); 输出？
  2. int i=0; do { printf("%d", ++i); } while(i<3); 输出？
  3. int a=3,b=4; printf("%d", a>b?a:b); 输出？
答案
  1. 1（&& 先，再 ||，真→1）
  2. 123
  3. 4
