一、单项选择题（本大题共 20 小题，每小题 3 分，共 60 分） ​

1. 以下关于 C 程序的叙述中，错误的是（ ） ​

A. 语句最后必须是分号
B. C 程序的基本组成单位是函数
C. 注释可以用 \N 开始
D. 一条语句可以分成多行来写

答案：C · 考点 1.1 C语言概述与基本概念

解析：C 语言注释用 /* */ 或 //，不存在 \N 开头的注释格式。A 正确（语句以分号结尾）；B 正确（C 以函数为基本组成单位）；D 正确（一条语句可跨多行）。

2. C 语言中，使用 strcmp 函数需要包含的头文件是（ ） ​

A. string.h B. math.h C. stdlib.h D. stdio.h

答案：A · 考点 2.4 串、数组和广义表 · 1.6 数组

解析：字符串处理函数（strcmp、strcpy、strlen、strcat 等）声明在 string.h。B 数学函数，C 标准库（malloc/free），D 标准输入输出。

3. 下面程序输出结果为（ ） ​

c
int a = 1, b = 0;
printf("%d", b = a + b);
printf("%d", a = 2 - b);
1
2
3

A. 1, 0 B. 1, 2 C. 3, 2 D. 0, 0

答案：B（回忆版共识） · 考点 1.2 数据的存储与运算

解析 / 注意：

  • 按字面逐步：b = a + b → b = 1，输出 1；a = 2 - b → a = 1，输出 1 → 连写为 11，选项中无。
  • 回忆版 OCR/措辞可能偏差：若第二式为 a = 2 * b，则输出 1 与 2，对应选项 B。
  • 刷题重点：赋值表达式可作 printf 实参；逗号/连写输出不要想当然。

4. 表达式 10 != 9 的值是（ ） ​

A. 0 B. 1 C. 8 D. 非 0

答案：B · 考点 1.2 数据的存储与运算

解析：!= 为不等于。10 != 9 为真，C 中关系表达式为真时值为 1（不是「任意非 0」）。

5. 以下表达式中值为 3 的是（ ） ​

A. 2 + 3 / 2
B. 17 - 14 % 10
C. 13 / 3 - 2
D. (3 + 6) / (13 - 9)

答案：A · 考点 1.2 数据的存储与运算

解析（整数除法截断）：

选项计算结果
A3/2=1，2+1=3✅ 3
B14%10=4，17-4=1313
C13/3=4，4-2=22
D9/4=22

6. 下列可用做 C 语言字符串常量的是（ ） ​

A. X B. $ABC C. "Horse" D. ¥abc

答案：C · 考点 1.1 C语言概述与基本概念

解析：字符串常量必须用双引号。A 为字符常量写法不完整；B、D 不是合法字符串常量写法。

7. 执行代码 int a = 205, b = 206; printf("%d", (a, b)); 的输出结果为（ ） ​

A. 205 B. 206 C. 205, 206 D. 206, 206

答案：B · 考点 1.2 数据的存储与运算

解析：(a, b) 是逗号表达式：从左到右计算，取最后一个表达式的值 → b 即 206。
对比：printf("%d", a, b) 格式串只有一个 %d，多出的参数行为未定义/被忽略，不是本题。

8. 以下程序段的功能是（ ） ​

c
int temp, x, y;
temp = x, x = y, y = temp;
1
2

A. 交换 x 和 y 的值
B. 交换 x 和 temp 的值
C. 交换 x 和 z 的值
D. x 和 y 从大到小排序

答案：A · 考点 1.3 顺序程序设计

解析：经典三变量交换；此处用逗号表达式把三条赋值写在一行。

9. 执行语句 int a = 1; while (a < 4) printf("%d", a++); 的输出结果为（ ） ​

A. 1 B. 12 C. 1234 D. 123

答案：D · 考点 1.5 循环结构程序设计

解析（a++ 后置：先用后加）：

a条件输出后 a
11<412
22<423
33<434
44<4 假退出

最终输出 123。

10. 下列关于算法的说法中，错误的是（ ） ​

A. 常用伪代码描述算法
B. 自然语言可以描述算法
C. 算法是求解问题的有限序列
D. 算法不存在时间复杂度

答案：D · 考点 2.9 算法基本概念与分析

解析：算法一定有时间复杂度（效率度量）。A/B/C 均为正确描述。

11. C 语言结构体类型变量在程序执行期间（ ） ​

A. 所有成员一直驻留在内存中
B. 只有一个成员驻留在内存中
C. 部分成员驻留在内存中
D. 没有成员驻留在内存中

答案：A · 考点 1.9 结构体与共用体

解析：结构体变量定义后，所有成员同时分配（含对齐）。联合体 union 才是成员共享同一块内存。

12. 长度为 n 的顺序表，在第 i 个元素后面插入一个新元素的时间复杂度为（ ） ​

A. O(0) B. O(1) C. O(n) D. O(n²)

答案：C · 考点 2.2 线性表

解析：插入需将 i 后元素整体后移；最坏约 n 次移动，平均 n/2 → O(n)。

13. 通过折半查找对关键字序列 {12, 18, 21, 35, 45, 55, 66}，查找 21 需要查找（ ）次 ​

A. 4 B. 3 C. 2 D. 1

答案：B · 考点 2.7 查找

解析（下标 0…6，已升序）：

  1. mid = (0+6)/2 = 3 → a[3]=35，21 < 35 → 左半
  2. mid = (0+2)/2 = 1 → a[1]=18，21 > 18 → 右半
  3. mid = (2+2)/2 = 2 → a[2]=21 找到

共 3 次。

14. 串 "abcd" 的长度为（ ） ​

A. 5 B. 2 C. 6 D. 4

答案：D · 考点 2.4 串、数组和广义表

解析：串长 = 字符个数，不含 '\0'。strlen("abcd")=4；sizeof("abcd")=5（含结束符）。

15. 数据的结构类型，从逻辑上可分为（ ） ​

A. 动态结构和静态结构
B. 紧凑结构和非紧凑结构
C. 线性结构和非线性结构
D. 内部结构和外部结构

答案：C · 考点 2.1 数据结构基本概念

解析：

  • 逻辑结构：线性（表/栈/队列/串） vs 非线性（树/图/集合）
  • 存储结构：顺序 / 链式 / 索引 / 散列

16. 链式存储的存储结构所占空间（ ） ​

A. 只有一部分存放结点值
B. 只有一部分存放结点间关系的指针
C. 分两部分，一部分存放结点值，另一部分存放结点所占单元数
D. 分两部分，一部分存放结点值，另一部分存放表示结点间关系的指针

答案：D · 考点 2.2 线性表

解析：结点 = 数据域 + 指针域。C 错在「所占单元数」——指针存的是邻接地址。

17. 带有头结点的单链表，头指针为 head，则表示单链表为空的选项是（ ） ​

A. head == NULL
B. head != NULL
C. head->next == NULL
D. head->next == head

答案：C · 考点 2.2 线性表

解析：带头结点时头结点始终存在；空表 ⟺ 头结点后无数据结点 ⟺ head->next == NULL。
A 是不带头结点判空；D 常见于循环链表等场景。

18. 四个元素 1、2、3、4 依次入栈，第一个出栈元素为 4，则第 3 个出栈元素为（ ） ​

A. 1 B. 2 C. 3 D. 4

答案：B · 考点 2.3 栈和队列

解析（LIFO）：四个全入后才弹 → 出栈序 4, 3, 2, 1 → 第 3 个是 2。

19. 二叉树第 3 层的结点个数最多有（ ）个 ​

A. 1 B. 4 C. 8 D. 16

答案：B · 考点 2.5 树和二叉树

解析：第 k 层最多 (2^{k-1}) 个（根为第 1 层）。第 3 层：(2^{2}=4)。

20. 在单链表中，要将 s 所指向结点插入到 p 所指向结点之后，其语句应为（ ） ​

A. s->next = p + 1; p->next = s;
B. s->next = p->next; p->next = s->next;
C. (*p).next->s; (*s).next = (*p).next;
D. s->next = p->next; p->next = s;

答案：D · 考点 2.2 线性表

解析（顺序不能反）：

  1. s->next = p->next; // 先挂上 p 的后继
  2. p->next = s; // 再让 p 指向 s

先改 p->next 会丢后继。B 第二步把 p->next 设回 s->next 等于没插对。

共同点：都是操作受限的线性表。区别：栈 LIFO，队列 FIFO。

串 = 零个或多个字符的有限序列。

9. 算法的时间复杂度和空间复杂度一定呈反比关系。（ ） ​

答案：× · 2.9 算法基本概念与分析

无必然反比；仅部分场景存在「以空间换时间」权衡。

10. 在线性表中，除了第一个元素外，每个元素有且只有一个直接前驱。（ ） ​

答案：√ · 2.2 线性表

线性表基本特性（首元无前驱，末元无后继）。

四、简答题（本大题共 4 小题，共 40 分） ​

1. 什么是数据项？什么是抽象数据类型（ADT）？（10 分） ​

考点 2.1 数据结构基本概念

参考答案：

  • 数据项：数据结构中讨论的最小单位，不可再分割。例：学生的学号、姓名、成绩。
  • 抽象数据类型（ADT）：一个数学模型 + 定义在该模型上的一组操作。特征：
    • 抽象性：只关心「做什么」，不关心「怎么做」
    • 封装性：数据与操作封装，外部仅通过接口访问
  • 常用三元组：ADT = (D, S, P) —— 数据对象、关系集、操作集
  • 例：整数 ADT 含数据 {…,-1,0,1,…} 与操作 {+,-,*,/,%}，使用者无需知道底层二进制实现。

参考答案：

  • 功能：计算数组 a 中 10 个元素的平均值并输出。
  • 循环结束时 i = 10（当 i=10 时 i<10 不成立退出）。

3. 简述顺序表存储结构的优缺点。（10 分） ​

考点 2.2 线性表

优点缺点
随机访问：下标 O(1)插入/删除平均 O(n)，需大量移动
存储密度高（无指针开销）预分配容量，易浪费或不够用
实现简单（数组即可）需连续大块内存，碎片敏感

适用：数据量已知、查多改少。

4. 什么是有向图？什么是有向图中顶点的出度和入度？（10 分） ​

考点 2.6 图

  • 有向图：顶点集 + 边集，边为有序对 <v,w>（从 v 指向 w 的弧）。<v,w> 与 <w,v> 是不同的边。
  • 出度 OD(v)：以 v 为弧尾的弧数（从 v 出发）。
  • 入度 ID(v)：以 v 为弧头的弧数（指向 v）。
  • 性质：Σ 出度 = Σ 入度 = 边数。

3. 编程：计算 1~300 之间所有奇数的和与偶数的和，并分别输出。 ​

考点 1.5 循环结构程序设计

c
#include <stdio.h>
int main() {
    int sum_odd = 0;
    int sum_even = 0;
    for (int i = 1; i <= 300; i++) {
        if (i % 2 == 1) {
            sum_odd += i;
        } else {
            sum_even += i;
        }
    }
    printf("奇数和 = %d\n", sum_odd);
    printf("偶数和 = %d\n", sum_even);
    return 0;
}
