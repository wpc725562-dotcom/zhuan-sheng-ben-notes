# 文件操作

> 🎯 **一句话秒杀**：文件操作就是"**打开→读写→关闭**"三步走，C语言用 `fopen` 开门、`fprintf/fscanf` 读写、`fclose` 关门。

---

## 一、文件的基本概念

### 文件分类

| 分类方式 | 类型 | 说明 |
|:---|:---|:---|
| 按存储方式 | 文本文件 | ASCII码存储，可读 |
| | 二进制文件 | 按内存原样存储，不可直接读 |
| 按操作方式 | 缓冲文件系统 | 自动在内存开辟缓冲区（标准C） |
| | 非缓冲文件系统 | 需程序自行管理缓冲区 |

> **C语言用的是缓冲文件系统**——系统自动在内存中为每个文件开辟缓冲区

### 文件指针

```c
FILE *fp;   // FILE是系统定义的结构体类型，fp是文件指针
```

## 二、打开文件 fopen

```c
FILE *fopen(文件名, 使用方式);
```

```c
FILE *fp = fopen("test.txt", "r");
if (fp == NULL) {          // ⚠️ 必须检查是否打开成功！
    printf("文件打开失败");
    return -1;
}
```

### 文件使用方式

| 模式 | 含义 | 文件不存在时 |
|:---|:---|:---:|
| `"r"` | 只读（文本） | 报错 |
| `"w"` | 只写（文本） | 新建 |
| `"a"` | 追加（文本） | 新建 |
| `"rb"` | 只读（二进制） | 报错 |
| `"wb"` | 只写（二进制） | 新建 |
| `"ab"` | 追加（二进制） | 新建 |
| `"r+"` | 读写（文本） | 报错 |
| `"w+"` | 读写（文本） | 新建 |

### 🚨 红牌警告
> `"w"` 和 `"w+"` 会**清空原有文件内容**！文件原来有东西也会被删光
> `"a"` 追加模式——不会清空文件，写的内容加到末尾

## 三、关闭文件 fclose

```c
fclose(fp);    // 关闭文件
fp = NULL;     // 指针置空（好习惯）
```

### 为什么一定要关闭？
1. 释放文件指针占用的资源
2. **刷新缓冲区**——确保所有数据真正写入磁盘
3. 程序能打开的文件数量有限

## 四、文件的读写

### 1. 字符读写：fgetc / fputc

```c
// 写一个字符
fputc('A', fp);

// 读一个字符
char ch = fgetc(fp);       // 读到文件末尾返回 EOF
```

### 2. 字符串读写：fgets / fputs

```c
// 写字符串
fputs("Hello", fp);

// 读字符串
char str[100];
fgets(str, 100, fp);       // 读一行（包括换行符），最多读99个字符
```

### 3. 格式化读写：fprintf / fscanf

```c
// 写入文件
fprintf(fp, "%d %s %.1f", 1001, "张三", 85.5);

// 从文件读取
int id; char name[20]; float score;
fscanf(fp, "%d %s %f", &id, name, &score);
```

### 🔑 秒杀秘籍
> `fprintf` 和 `fscanf` 和 `printf`/`scanf` 用法一样——只是多了个 `FILE *fp` 参数

### 4. 数据块读写：fread / fwrite（重点！）

```c
size_t fread(void *ptr, size_t size, size_t count, FILE *fp);
size_t fwrite(const void *ptr, size_t size, size_t count, FILE *fp);
```

```c
// 将结构体数组写入文件
struct Student stu[3] = {...};
fwrite(stu, sizeof(struct Student), 3, fp);

// 从文件读取回结构体数组
struct Student stu[3];
fread(stu, sizeof(struct Student), 3, fp);
```

### 🚨 红牌警告
> **fread/fwrite 用于二进制文件**——读写速度快，但不能直接查看内容
> **fprintf/fscanf 用于文本文件**——可读性好，但速度慢

## 五、文件定位

| 函数 | 功能 |
|:---|:---|
| `rewind(fp)` | 将文件位置指针**重置到文件开头** |
| `fseek(fp, 偏移量, 起始点)` | 将位置指针移动到指定位置 |
| `ftell(fp)` | 返回当前指针位置（距文件开头字节数） |
| `feof(fp)` | 判断是否到文件末尾 |

### fseek 的起始点

```c
fseek(fp, 10, SEEK_SET);   // 从文件开头偏移10字节
fseek(fp, -5, SEEK_END);   // 从文件末尾倒退5字节
fseek(fp, 3, SEEK_CUR);    // 从当前位置偏移3字节
```

| 起始点 | 常量名 | 值 |
|:---|:---:|:---:|
| 文件开头 | `SEEK_SET` | 0 |
| 当前位置 | `SEEK_CUR` | 1 |
| 文件末尾 | `SEEK_END` | 2 |

### 完整文件操作示例

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    // 1. 打开文件
    FILE *fp = fopen("data.txt", "w");
    if (fp == NULL) {
        printf("打开失败");
        exit(1);
    }

    // 2. 写入数据
    fprintf(fp, "Hello, File!\n");
    fprintf(fp, "1001 张三 85.5\n");

    // 3. 关闭
    fclose(fp);

    // 4. 重新打开读取
    fp = fopen("data.txt", "r");
    if (fp == NULL) {
        printf("打开失败");
        exit(1);
    }

    // 5. 读取
    char str[100];
    while (fgets(str, 100, fp) != NULL) {
        printf("%s", str);
    }

    // 6. 关闭
    fclose(fp);
    return 0;
}
```

### 检测文件末尾

```c
// 方式1：feof
while (!feof(fp)) {
    // 读数据...
}

// 方式2：fgets（读到末尾返回NULL）
while (fgets(str, 100, fp) != NULL) {
    // 处理每一行...
}

// 方式3：fgetc（读到末尾返回EOF）
char ch;
while ((ch = fgetc(fp)) != EOF) {
    // 处理每个字符...
}
```

```
3. fread和fwrite通常用于什么类型的文件？

4. rewind(fp) 的作用？

5. feof(fp) 的作用？
```

<details>
<summary>👆 点击展开答案</summary>

1. **文件打开失败**（例如文件不存在时用"r"模式打开）
2. **"w"** 清空原内容后写入；**"a"** 保留原内容，在文件末尾追加
3. **二进制文件**（fread/fwrite按数据块读写，速度快）
4. **将文件位置指针重置到文件开头**
5. **判断是否到达文件末尾**（到末尾返回非0）
