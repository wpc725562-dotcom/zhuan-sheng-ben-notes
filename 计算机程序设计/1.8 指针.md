# 指针 ⭐

> 🎯 **一句话秒杀**：指针就是**地址**——指针变量是存储地址的变量。`*` 是"取出指针指向的内容"，`&` 是"取变量的地址"。

---

## 一、什么是指针？

```c
int a = 10;     // 普通变量：存值
int *p = &a;    // 指针变量：存地址
```

```
内存模型：
变量 a: [10 ]  ← 地址 0x1000
指针 p: [0x1000]  ← 指向 a（存储的是a的地址）
```

```c
printf("%d", a);   // 10 — 直接访问
printf("%d", *p);  // 10 — 间接访问（通过地址找到a的值）
```

## 二、指针的定义与使用

```c
int *p;       // p是指向int的指针
char *cp;     // cp是指向char的指针
double *dp;   // dp是指向double的指针

int a = 10;
p = &a;       // p指向a（p存a的地址）
*p = 20;      // 通过p修改a的值（a变成20）
```

### 🚨 红牌警告
> **定义时 `*` 是类型的一部分：** `int *p;` 表示 `p` 是 `int*` 类型
> **使用时 `*` 是间接访问运算符：** `*p` 表示取p指向的值
> **没有初始化的指针不能用！** 野指针会导致程序崩溃

## 三、指针的运算

| 运算 | 含义 | 示例 |
|:---|:---|:---|
| `p + n` | 指向后面第n个元素 | `p + 1` 指向下一个int（地址+4） |
| `p - n` | 指向前面第n个元素 | `p - 1` 指向上一个元素 |
| `p1 - p2` | 两个指针间的元素个数 | |
| `*p` | 取内容 | |
| `&a` | 取地址 | |

### 🔑 秒杀秘籍——指针加1加多少？

```c
int *p;     // p+1 → 地址+4（一个int 4字节）
char *cp;   // cp+1 → 地址+1（一个char 1字节）
double *dp; // dp+1 → 地址+8（一个double 8字节）
```

> **指针+1 = 地址 + sizeof(指向的类型) × 1**
> 所以指针加1是"跳到下一个元素"，不是"地址加1"

## 四、指针与数组（重点！）

### 数组名就是首元素地址

```c
int a[5] = {10, 20, 30, 40, 50};
int *p = a;     // p指向a[0]，等价于 int *p = &a[0]
```

### 通过指针访问数组元素

```c
// 以下四种方式完全等价！
a[0]        // 下标法
*a          // 指针法
p[0]        // 指针当下标用
*(p+0)      // 指针偏移

// 一般情况：a[i] ≡ *(a+i) ≡ *(p+i) ≡ p[i]
```

### 遍历数组

```c
int a[5] = {10, 20, 30, 40, 50};
int *p;

// 方式1：下标
for (int i = 0; i < 5; i++)
    printf("%d ", a[i]);

// 方式2：指针
for (p = a; p < a + 5; p++)
    printf("%d ", *p);
```

### 🚨 红牌警告
> **数组名是常量指针——不能修改！**
> ```c
> int a[5];
> a++;      // ❌ 编译错误！a是常量
>
> int *p = a;
> p++;      // ✅ 指针变量可以修改
> ```

### 数组名作为函数参数

实际上传递的是指针：

```c
// 这三个声明完全等价
void func(int arr[]);
void func(int arr[10]);  // 写的10没用
void func(int *arr);

// 函数内部：
void func(int arr[]) {
    int n = sizeof(arr) / sizeof(arr[0]);  // ❌ 错误！
    // sizeof(arr) 在这里是4（指针大小），不是数组大小！
}
```

## 五、指针与函数

### 用指针实现交换

```c
void swap(int *a, int *b) {
    int temp = *a;     // 取出a指向的值
    *a = *b;           // 把b指向的值赋给a指向的位置
    *b = temp;         // temp赋给b指向的位置
}
// 调用：swap(&x, &y);  → x和y被交换 ✅
```

### 指针作为函数返回值

```c
int *find_max(int *arr, int n) {
    int *max = arr;
    for (int i = 1; i < n; i++)
        if (arr[i] > *max) max = &arr[i];
    return max;
}
// 调用：int *p = find_max(a, 5);
```

### 🚨 红牌警告
> **不能返回局部变量的地址！**
> ```c
> int *bad() {
>     int a = 10;
>     return &a;  // ❌ a是局部变量，函数结束就销毁
> }
> ```

## 六、动态内存分配

| 函数 | 原型 | 功能 |
|:---|:---|:---|
| `malloc` | `void *malloc(size_t size)` | 分配size字节内存 |
| `calloc` | `void *calloc(n, size)` | 分配n×size字节并清零 |
| `realloc` | `void *realloc(void *ptr, size_t size)` | 重新分配大小 |
| `free` | `void free(void *ptr)` | 释放内存 |

```c
#include <stdlib.h>

int *p = (int *)malloc(5 * sizeof(int));  // 分配5个int的空间
if (p == NULL) {                          // 检查是否分配成功
    printf("内存不足");
    return -1;
}
for (int i = 0; i < 5; i++)
    p[i] = i * 10;
free(p);      // ⚠️ 用完后必须释放！
p = NULL;     // 避免野指针
```

### 🚨 红牌警告
> **malloc 三要素：**
> 1. `malloc` 返回 `void*` — 需要**强制类型转换**为需要的类型
> 2. **检查是否返回 NULL**（内存不足时返回NULL）
> 3. **用完必须 free**，否则内存泄漏

## 📝 闭卷真题挑战

**（点击下方空白查看答案）**

```
1. int a[5]; 则 &a[0] 和 a 的关系是？

2. 写出输出：
   int a[5]={10,20,30,40,50}, *p=a;
   p++; printf("%d", *p);

3. 指针+1，地址增加多少？为什么？

4. 为什么不能返回局部变量的地址？

5. malloc 后忘记 free 会怎样？
```

<details>
<summary>👆 点击展开答案</summary>

1. **完全相等**（a就是首元素地址 &a[0]）
2. **20**（p先++指向a[1]，再取内容）
3. 取决于指针类型：`int* +1` → 地址+4，`char* +1` → 地址+1。因为指针+1是"跳到下一个元素"
4. **局部变量在函数返回后被销毁**，返回的地址指向无效内存
5. **内存泄漏**（memory leak）——程序占用的内存越来越多

</details>
