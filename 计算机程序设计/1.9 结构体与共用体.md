# 结构体与共用体

> 🎯 **一句话秒杀**：结构体是把不同类型的数据打包成一个新类型；共用体是让不同类型共享同一块内存（改一个全变）。

---

## 一、结构体的定义

```c
struct 结构体名 {
    类型 成员1;
    类型 成员2;
    ...
};   // ⚠️ 这里的分号不能少！
```

```c
// 定义学生类型
struct Student {
    int id;           // 学号
    char name[20];    // 姓名
    float score;      // 成绩
};  // ← 分号！
```

## 二、结构体变量的定义

```c
// 方式1：先定义结构体类型，再定义变量
struct Student stu1, stu2;

// 方式2：定义类型的同时定义变量
struct Student {
    int id;
    char name[20];
    float score;
} stu1, stu2;

// 方式3：直接定义变量（省略结构体名）
struct {
    int id;
    char name[20];
    float score;
} stu1, stu2;
```

### 初始化

```c
struct Student stu1 = {1001, "张三", 85.5};
struct Student stu2 = {1002, "李四", 92.0};
```

## 三、结构体成员的访问

### 通过成员运算符 `.`

```c
struct Student stu;
stu.id = 1001;              // 赋值
strcpy(stu.name, "张三");   // 字符串成员赋值（不能用=）
stu.score = 85.5;
printf("%d %s %.1f", stu.id, stu.name, stu.score);
```

### 🚨 红牌警告
> **结构体中的字符数组不能直接用 `=` 赋值！**
> ```c
> stu.name = "张三";  // ❌ 错误！字符数组不能这样赋
> strcpy(stu.name, "张三");  // ✅ 用strcpy
> ```

## 四、结构体与指针

### 结构体指针

```c
struct Student stu = {1001, "张三", 85.5};
struct Student *p = &stu;

// 以下三种方式等价
stu.score = 90;        // 直接访问
(*p).score = 90;       // 括号不能省！
p->score = 90;         // 用 -> 最常用 ✅
```

### 🔑 秒杀秘籍
> `p->成员` 等价于 `(*p).成员`——**箭头指向运算符**专门用于结构体指针
> 括号不能省：`*p.score` 等价于 `*(p.score)` — 但 `p.score` 是错的（p是指针不是结构体变量）

## 五、结构体做函数参数

```c
// 值传递（整个结构体拷贝，效率低）
void print_stu(struct Student s) {
    printf("%d %s %.1f", s.id, s.name, s.score);
}

// 指针传递（只传地址，效率高 ✅）
void print_stu(struct Student *s) {
    printf("%d %s %.1f", s->id, s->name, s->score);
}
```

### 🚨 红牌警告
> **结构体做参数时，值传递会拷贝整个结构体！** 结构体很大时非常浪费内存和CPU
> 建议**大规模用指针传递，小规模用值传递**

## 六、结构体数组

```c
struct Student class[3] = {
    {1001, "张三", 85.5},
    {1002, "李四", 92.0},
    {1003, "王五", 78.0}
};

// 访问
printf("%s", class[0].name);   // 张三
class[1].score = 95.0;          // 修改
```

## 七、共用体（联合体）

```c
union Data {
    int i;
    float f;
    char c;
};
```

### 关键特性：所有成员共享同一块内存

```c
union Data d;
d.i = 10;            // 内存中存放 10
printf("%d", d.i);   // 10 ✅
printf("%f", d.f);   // ❌ 乱码（因为d.f和d.i共享同一块内存）
```

| 特点 | struct（结构体） | union（共用体） |
|:---|:---|:---|
| 内存分配 | **各成员独立存储** | **所有成员共享同一块内存** |
| 总大小 | 各成员大小**之和**（+对齐） | **最大成员的大小** |
| 同时存储 | 所有成员**共存** | **只能存一个** |
| 用途 | 组织不同类型的数据 | 节省空间/多选一 |

```c
struct S { char c; int i; };   // sizeof = 8（含对齐）
union U { char c; int i; };    // sizeof = 4（取int的大小）
```

## 八、typedef——给类型起别名

```c
typedef struct Student {
    int id;
    char name[20];
    float score;
} Stu;   // Stu 就是 struct Student 的别名

// 现在可以直接写：
Stu stu1;                // 等价于 struct Student stu1;
Stu *p;                  // 结构体指针
Stu class[30];           // 结构体数组
```

### 不用typedef的对比

```c
// 没有typedef
struct Student stu1;             // 每次都要写 struct

// 有typedef
typedef struct {                 // 甚至可以省略结构体名
    int id;
    char name[20];
    float score;
} Stu;
Stu stu1;                        // 简洁！
```
