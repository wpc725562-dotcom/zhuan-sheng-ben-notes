## 一、单项选择题

1. $\lim_{x\to0}(2^{x}+1)$ → **C. 2**
2. $f(x)=(1+x^{2})^{1/x^{2}}\ (x\neq0)$ 在 0 连续，$a=$ → **C. $e$**
3. $y=xe^{-x}$ 在 $(1,e^{-1})$ 斜率 → **B. 0**
4. $2x$ 是 $f$ 原函数，$\int_0^{\pi/2}[f-\sin x]$ → **A. $\pi-1$**
5. $\sum u_n$ 收敛，下列**发散** → **D. $\sum(u_n-1)$**

## 二、填空

6. $y=\sin x/x$ 水平渐近线 $y=$ **0**
7. $\int_k^{+\infty}x^{-2}dx=1$，$k=$ **1**
8. $z=x^{x}+(x-y)^{2}$，$\partial^{2}z/\partial y\partial x=$ **$-2$**
9. 换序 $\int_0^1 dx\int_{\sqrt x}^1 f\,dy$ → $\int_0^1 dy\int_0^{y^{2}} f\,dx$
10. $y''-8y'+7y=0$ 通解 → **$C_1 e^{x}+C_2 e^{7x}$**

## 三、计算

11. $\lim_{x\to2}\dfrac{x^{3}+3x^{2}-20}{x^{2}-4}$ → **6**
12. $y=\sqrt{x+\cos x}$，$dy|_{x=0}$ → $\frac12 dx$
13. $f'=\ln x/x$，凹 $(0,e)$，凸 $(e,+\infty)$
14. $\int(2x-1)e^{x}dx$ → $(2x-3)e^{x}+C$
15. 分段 $f$，$\int_{-1}^{4}f$ → **3**
16. $z=\ln3-x^{2}e^{y^{2}z}$ 求偏导（见标准答案）
17. $\iint_D(x^{2}+y^{2})^{2}$，$x^{2}+y^{2}\le1$ → $\pi/3$
18. $(1/2)^n\le u_n\le n^{2}/2^{n}$，$\sum u_n$ → **收敛**

## 四、综合

19. 证：(1) $\arctan x=\pi/2-\arctan(1/x)$ (2) $\arctan x<\ln(x+\sqrt{1+x^{2}})$
20. $f\ge\sqrt{1+x^{4}}$，夹面积 $t^{3}$ → $f=\sqrt{1+x^{4}}+3x^{2}$；
    $fg'+f'g=5x\sqrt x$，$g(0)=1$ → $g=\dfrac{2x^{5/2}+1}{\sqrt{1+x^{4}}+3x^{2}}$
