# 高等数学 · 2018 年真题演练

> 类型：同型演练（B/C 级资料）| 主攻：**极限与连续**
> 边界：完整原卷以正版真题册为准。见 [[00-资料来源与使用说明]]

## 题目

### 1. 【选择题】

$\lim_{x\to 0}\frac{\sin 3x}{x}=$

**答案：** C. 3
**讲解：** 化为 (sin3x)/(3x)*3 → 3

### 2. 【选择题】

$f(x)=\frac{x^2-1}{x-1}$ 在 $x=1$

**答案：** B. 可去间断
**讲解：** 化简为 x+1，极限存在但点无定义

### 3. 【计算】

求 $\lim_{x\to 0}\frac{1-\cos x}{x^2}$

**答案：** $1/2$
**讲解：** 1-cos x 等价于 x^2/2

### 4. 【计算】

求 $y=\ln(x+\sqrt{x^2+1})$ 的导数

**答案：** $1/\sqrt{x^2+1}$
**讲解：** 链式法则

### 5. 【计算】

讨论 $f(x)=x\sin(1/x)$（x≠0），f(0)=0 在 0 的连续性

**答案：** 连续
**讲解：** 夹逼 |x sin(1/x)| ≤ |x| → 0

## 得分策略

- 极限：等价无穷小优先，再洛必达
- 导数：复合 / 隐函数 / 参数方程分型训练
- 积分：先看是否凑微分，再分部
