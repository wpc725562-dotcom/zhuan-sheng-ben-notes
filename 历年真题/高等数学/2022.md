## 一、单项选择题

1. $f(x)=\begin{cases}x+1,&x\neq1\\a,&x=1\end{cases}$ 在 1 连续 → **D. $a=2$**
2. $\lim_{x\to0}(1-3x)^{1/x}$ → **A. $e^{-3}$**
3. $\lim u_n=0$ 是级数收敛的 → **B. 必要条件**
4. $1/x^{2}$ 是 $f$ 原函数，$\int_1^{+\infty}f$ → **C. $-1$**
5. $I=\int_0^1 dx\int_x^1 f(x^{2}+y^{2})dy$ 极坐标 → **D.** $\int_{\pi/4}^{\pi/2}d\theta\int_0^{\csc\theta} r f(r^{2})dr$

## 二、填空

6. $2x$ 与 $3x^{2}+mx$ 等价 → $m=$ **2**
7. $\begin{cases}x=5t-t^{2}\\y=\log_2 t\end{cases}$，$dy/dx|_{t=2}=$ **$1/(2\ln2)$**
8. 椭圆 $x^{2}/4+y^{2}/3=1$ 绕 $x$ 轴体积 → **$8\pi$**
9. $e^{-x}y'=2$ 通解 → **$y=2e^{x}+C$**
10. $z=x^{\ln y}$ 在 $(e,e)$ 全微分 → **$dx+dy$**

## 三、计算

11. $\lim_{x\to1}\dfrac{x^{3}+3x^{2}-9x+5}{x^{3}-3x+2}$ → **2**
12. $y=\arctan x^{2}$，$y''|_{x=1}$ → **$-1$**
13. $f(x)=x^{2}\sin(1/x)+2x\ (x\neq0)$，$f'(0)$ → **2**
14. $\int\dfrac{2x^{2}+3x}{x\sqrt{1-x^{2}}}dx$ → $3\arcsin x-2\sqrt{1-x^{2}}+C$
15. $\int_0^{\pi/4}x\sec^{2}x\,dx$ → $\pi/4-\frac12\ln2$
16. $z=2x-y^{2}e^{z}$，$z_x-y z_y$ → **2**
17. $\iint_D\cos x\,d\sigma$（$y=\sin x$ 等围成）→ **$1/2$**
18. $\sum(n/3^{n}-3/2^{n})$ → **收敛**

## 四、综合

19. $f=2x\ln x-x-1/x+2$：拐点 $(1,0)$；**不存在**过原点切线
20. (1) 证 $\int_0^x f(x-t)dt=\int_0^x f(t)dt$
    (2) $f=3x+1+\int_0^x t f-\,x\int_0^x f(x-t)$ → $f=\cos x+3\sin x$
