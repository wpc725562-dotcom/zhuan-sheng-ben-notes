# 高等数学 · 2022 年真题演练

## 题目

### 1. 【选择】

$z=x^2 y$ 的偏导 ∂z/∂x

**答案：** 2xy
**讲解：** 视 y 为常数

### 2. 【计算】

解 y'=2xy

**答案：** y=C e^{x^2}
**讲解：** 可分离变量

### 3. 【计算】

一阶线性 y'+y=e^x

**答案：** y=(1/2)e^x + C e^{-x}
**讲解：** 通解公式

### 4. 【计算】

f(x,y)=x^2+y^2 在 (0,0) 的极值

**答案：** 极小值 0
**讲解：** AC-B² 大于 0 且 A 大于 0

### 5. 【概念】

全微分存在的充分条件

**答案：** 偏导连续
**讲解：** 充分非必要

## 得分策略

- 极限：等价无穷小优先，再洛必达
- 导数：复合 / 隐函数 / 参数方程分型训练
- 积分：先看是否凑微分，再分部
