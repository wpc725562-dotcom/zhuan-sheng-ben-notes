# 高等数学 · 2021 年真题演练

> 类型：同型演练（B/C 级资料）| 主攻：**中值定理与洛必达**
> 边界：完整原卷以正版真题册为准。见 [[00-资料来源与使用说明]]

## 题目

### 1. 【选择】

罗尔定理中 f(a)=f(b) 的作用

**答案：** 保证存在 ξ 使 f'(ξ)=0
**讲解：** 三条件缺一不可

### 2. 【计算】

$\lim_{x\to 0}\frac{\tan x-x}{x^3}$

**答案：** 1/3
**讲解：** 洛必达或泰勒

### 3. 【计算】

$\lim_{x\to+\infty}\frac{\ln x}{x}$

**答案：** 0
**讲解：** ∞/∞ 型洛必达

### 4. 【证明思路】

拉格朗日中值定理几何意义

**答案：** 割线平行于某切线
**讲解：** 存在 ξ 属于 (a,b)

### 5. 【计算】

求 $y=\sin x / x$ 在 x→0 的极限

**答案：** 1
**讲解：** 重要极限

## 得分策略

- 极限：等价无穷小优先，再洛必达
- 导数：复合 / 隐函数 / 参数方程分型训练
- 积分：先看是否凑微分，再分部
