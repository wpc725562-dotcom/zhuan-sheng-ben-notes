## 推荐刷法

```
1. 近 3 年（2022–2024）整套限时
2. 每周默写 2 篇写作（书信 + NOTICE）
3. 完形/阅读错题进单词本
4. 再补 2018–2021 → 2013–2017 → 更早
```

## 近年趋势

- 写作稳在**应用文**，议论文极少
- 客观题词汇贴近高职课标约 3000 词
- 默写时控制 **90–110 词**
- OCR 题干可能粘连/缺空格，**以本地 PDF 为准**
