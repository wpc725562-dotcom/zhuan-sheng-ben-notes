## 手撕模板

### 指针返回两数和
```c
void add(int x, int y, int *z) {
    *z = x + y;
}
```

### 过滤字符
```c
// 把 s 中 != ch 的拷到 t
int j = 0;
for (int i = 0; s[i]; i++)
    if (s[i] != ch) t[j++] = s[i];
t[j] = '\0';
```

### 二维按行填数
```c
int num = start;
for (int i = 0; i < n; i++)
    for (int j = 0; j < m; j++)
        a[i][j] = num++;
```
