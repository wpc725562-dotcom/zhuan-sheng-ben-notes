## 简答
**1. 实参与形参的传递方式（值传递）特点？**
形参改变一般不影响实参（指针/地址传递可改实参对象）。

**2. 二叉树先序/中序/后序遍历定义。**
先根；中根；后根。给出一棵小树写三序列。

## 编程骨架
写函数：求数组最大值。
```c
int max_of(int a[], int n){
    int m=a[0], i;
    for(i=1;i<n;i++) if(a[i]>m) m=a[i];
    return m;
}
```
