# 计算机程序设计 · 2021 年真题演练

> 主攻：**查找与排序** | 同型演练


## 简答
**折半查找的前提？时间复杂度？**
有序顺序表；平均 \(O(\log n)\)。

**常见排序稳定性与复杂度（默写表）：**

| 排序 | 平均时间 | 稳定? |
|:---|:---|:---:|
| 冒泡 | O(n²) | 是 |
| 简单选择 | O(n²) | 否 |
| 直接插入 | O(n²) | 是 |
| 快速 | O(n log n) | 否 |
| 归并 | O(n log n) | 是 |
| 堆 | O(n log n) | 否 |
