## 一、选择题（回忆要点）

1. 不能做标识符 → **C. `&a`**
2. `fopen` 失败 → **B. `NULL`**
3. 取第 i 结点及前驱最省时 → **C. 顺序表**
4. `a=4,b=3,c=2` 为真 → **D. `a&&b||c`**
5. 算法分析两方面 → **D. 时间 + 空间复杂度**
6. 结构体错误说法 → **B. 由系统直接提供**
7. 选存储结构不考虑 → **B. 各结点值如何**
8. `scanf` 正确 → **C. `scanf("%f%f",&c,&d)`**
9. `b=*(p++)`，`p=&a[1]`，`a[5]={10..50}` → **A. 20**
10. 栈特点 → **C. LIFO**
11. `for(int i=1;i++<4;);` 次数 → **A. 3**
12. 完全二叉树 12 结点叶子 → **B. 6**
13. 操作链表须找到 → **D. 表头**
14. 删 `q`（`p` 前驱）→ **`p->next=q->next; free(q);`**
15. `person` 是变量不是类型名
16. 双重 for → **O(n²)**
17. 二分查找 → **键值有序顺序表**
18. 字符串连接 → **`strcat`**
19. 树深度错误：各结点度最大值
20. `int a[5];` 输入 → **`scanf("%d",a)`**

## 三、填空（回忆）

1. `char` 占 **1** 字节
2. 图：元素 **多对多**
3. `a[][3]={0},{1},{2}`，`a[1][2]=` **0**
4. 图遍历：深度优先 + **广度**优先
5. 合并升序 5 与 9：最少比较约 **5**（最坏 13，以教材为准）

## 六、应用题（默写）

```c
// 1. 三次成绩取最大
int a,b,c,m; scanf("%d%d%d",&a,&b,&c);
m=a; if(b>m)m=b; if(c>m)m=c; printf("%d\n",m);

// 2. 正负零
int x; scanf("%d",&x);
if(x>0) puts("正数"); else if(x<0) puts("负数"); else puts("零");

// 3. 十班人数查 y
int a[10],y,i,ok=0;
for(i=0;i<10;i++) scanf("%d",&a[i]);
scanf("%d",&y);
for(i=0;i<10;i++) if(a[i]==y){ ok=1; break; }
puts(ok?"Success":"Fail");
```
