## 一、考试是什么

| 项目 | 说明 |
|:---|:---|
| 正式名称 | **广东省普通高等学校专升本招生考试**（俗称：普通专升本 / 统招专升本） |
| 组织方 | 广东省教育考试院统筹；命题与大纲以**当年度招生考试文件**为准 |
| 公共课常见科目 | 政治理论、英语、高等数学（按报考类别选考） |
| 专业课（本库已覆盖） | **计算机程序设计**（C 语言 + 数据结构）等 |
| 本库覆盖学科 | 公共英语 · 高等数学 · 政治理论 · 计算机程序设计 |

## 二、资料来历（务必读）

本库**不是**考试院官方原卷 PDF 的全文镜像，原因：

1. **版权**：完整官方试卷、标准答案的**全文复制与传播**受著作权法限制，个人笔记库不宜整卷粘贴。  
2. **保密与版本**：部分年份卷面仅在正版真题册 / 培训机构授权资料中流通，网络流传版本常有**题干改写、答案差错**。  
3. **考纲变动**：题型分值以**当年考试大纲**为准，跨年会微调。

### 本库采用的三级来源

| 层级 | 内容 | 可信度 | 在本库中的位置 |
|:---|:---|:---|:---|
| **A. 官方公开信息** | 省考试院 / 高校发布的**考试说明、科目设置、题型分值、参考书目** | ★★★★★ | 各科「索引」中的试卷结构表 |
| **B. 历年真题题型与考点复现** | 根据多年考生回忆卷、正版真题册公开目录、辅导书标注的**高频体裁与考点**，整理为**同型演练题** | ★★★★ | 各年「题目方向 / 考点」+ 答案讲解 |
| **C. 训练用自拟题** | 严格按广东题型结构仿写的练习题，用于默写与限时训练 | ★★★ | 标注为「演练题」的部分 |

### 建议你本地再备的「一手原卷」

| 渠道 | 用途 |
|:---|:---|
| 广东省教育考试院官网 `eea.gd.gov.cn` | 当年简章、科目、政策 |
| 正版《广东普通专升本历年真题汇编》类图书（天一 / 华图 / 金榜等，以**最新版**为准） | **完整原题 + 标准答案** |
| 报考院校公布的专业课大纲 / 样题 | 计算机等专业课边界 |

> **对照原则**：本库题目若与你手里某套纸质卷措辞不完全一致，**以纸质官方/正版册为准**；本库保证的是**广东题型结构 + 得分逻辑 + 可默写范文/解法**。

## 三、目录怎么用

```
历年真题/
├── 00-资料来源与使用说明.md    ← 你在这里
├── 总索引.md                    ← 全学科导航
├── 公共英语/                    ← 按年子文件
├── 高等数学/
├── 政治理论/
└── 计算机程序设计/
```

每个学科文件夹内：

- `_索引.md`：题型分值、备考顺序、年份一览  
- `YYYY.md`：该年**题目方向 + 参考答案/范文 + 得分点讲解**  
- 英语写作另可跳转：[[历年真题分类与讲解]]（原英语笔记库，已合并链接）
