# 3.2 直接积分法

> **学科系统**：高等数学 → 一元函数积分学 → 直接积分法
> **秒杀类比**：直接积分就像炒菜的"基本刀工"——先把菜切成标准形状（利用性质和公式变形），然后直接下锅（套积分公式）。

## 一、 核心知识解构

### 1. 什么是直接积分法
利用不定积分的**线性性质**和**基本积分公式**，对被积函数进行恒等变形（展开、拆项、化简等），然后逐项积分的方法。

### 2. 基本积分公式（必背！）

| 公式 | 结果 |
|:---|:---|
| $\int kdx$ | $kx + C$ |
| $\int x^\alpha dx$ | $\frac{x^{\alpha+1}}{\alpha+1} + C \; (\alpha \neq -1)$ |
| $\int \frac{1}{x}dx$ | $\ln|x| + C$ |
| $\int e^x dx$ | $e^x + C$ |
| $\int a^x dx$ | $\frac{a^x}{\ln a} + C$ |
| $\int \sin x dx$ | $-\cos x + C$ |
| $\int \cos x dx$ | $\sin x + C$ |
| $\int \frac{1}{1+x^2}dx$ | $\arctan x + C$ |
| $\int \frac{1}{\sqrt{1-x^2}}dx$ | $\arcsin x + C$ |

## 二、 常用变形技巧

| 题型 | 变形方法 | 示例 |
|:---|:---|:---|
| 多项式乘开 | 展开成多项式 | $\int(x+1)^2 dx = \int(x^2+2x+1)dx$ |
| 假分式 | 多项式除法化真分式 | $\int\frac{x^2}{x+1}dx = \int(x-1+\frac{1}{x+1})dx$ |
| 三角函数平方 | 降幂公式 | $\int\sin^2 x dx = \int\frac{1-\cos2x}{2}dx$ |
| 根式 | 化为指数形式 | $\int\sqrt[3]{x^2}dx = \int x^{2/3}dx$ |

## 三、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：$\int \frac{1}{x}dx = \ln|x| + C$ **绝对值不能丢！**
* 🚨 **易错雷区**：$\int \frac{1}{x^2}dx = -\frac{1}{x} + C$，不是 $\ln x^2$！
* 🔑 **秒杀秘籍**：遇到 $\int \frac{x^2}{1+x^2}dx$，加减1技巧：$\frac{x^2}{1+x^2} = 1 - \frac{1}{1+x^2}$，秒出答案 $x - \arctan x + C$。

## 四、 闭卷真题挑战

> **【真题演练】**：求 $\int \frac{x^4}{1+x^2}dx$。

> [!success]- 点击查看答案与解析
> **【正确答案】**：$\frac{x^3}{3} - x + \arctan x + C$
>
> **【核心解析】**：
> 用多项式除法：$\frac{x^4}{1+x^2} = x^2 - 1 + \frac{1}{1+x^2}$
> 所以：$\int\frac{x^4}{1+x^2}dx = \int(x^2 - 1 + \frac{1}{1+x^2})dx = \frac{x^3}{3} - x + \arctan x + C$
