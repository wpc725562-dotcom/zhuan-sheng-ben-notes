# 3.8 牛顿-莱布尼兹公式及定积分的性质

> **学科系统**：高等数学 → 一元函数积分学 → 牛顿-莱布尼兹公式
> **秒杀类比**：牛顿-莱布尼兹公式是积分学的"核武器"——它告诉你不需要每次做黎曼和，只用找一个原函数，代上下限相减即可！

## 一、 核心知识解构

### 1. 牛顿-莱布尼兹公式（微积分基本定理）
若 $f(x)$ 在 $[a,b]$ 上连续，$F(x)$ 是 $f(x)$ 的一个原函数，则：
$$ \boxed{\int_a^b f(x)dx = F(b) - F(a)} $$

### 2. 定积分的基本性质

| 性质 | 公式 |
|:---|:---|
| 线性性 | $\int_a^b [\alpha f + \beta g]dx = \alpha\int_a^b f dx + \beta\int_a^b g dx$ |
| 区间可加性 | $\int_a^b f dx = \int_a^c f dx + \int_c^b f dx$ |
| 上下限互换 | $\int_a^b f dx = -\int_b^a f dx$ |
| 上下限相等 | $\int_a^a f dx = 0$ |
| 保号性 | $f(x) \ge 0 \Rightarrow \int_a^b f(x)dx \ge 0$ |
| 估值不等式 | $m(b-a) \le \int_a^b f dx \le M(b-a)$ |

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：使用牛顿-莱布尼兹公式前，必须先确认 $f(x)$ 在 $[a,b]$ 上连续！如果 $f$ 有间断点，需要分段积分。
* 🚨 **易错雷区**：代入 $F(b) - F(a)$ 时，先代上限 $b$ 再减下限 $a$，顺序不能反！
* 🔑 **秒杀秘籍**：利用奇偶性可以简化定积分——$f$ 为奇函数且在对称区间上积分为 0；$f$ 为偶函数则为 2 倍半区间积分。

## 三、 闭卷真题挑战

> **【真题演练】**：求 $\int_0^{\frac{\pi}{2}} \sin x dx$。

> [!success]- 点击查看答案与解析
> **【正确答案】**：$1$
>
> **【核心解析】**：
> $\sin x$ 的一个原函数是 $-\cos x$。
> $\int_0^{\frac{\pi}{2}} \sin x dx = [-\cos x]_0^{\frac{\pi}{2}} = (-\cos\frac{\pi}{2}) - (-\cos 0) = 0 - (-1) = 1$
