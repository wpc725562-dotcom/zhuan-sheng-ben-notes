# 3.1 原函数与不定积分的概念及性质

> **学科系统**：高等数学 → 一元函数积分学 → 原函数与不定积分
> **秒杀类比**：微分是"把东西打碎"，积分是"把碎片拼回去"。原函数就是那个拼回去的完整物件——已知碎片找原物。

## 一、 核心知识解构

### 1. 原函数的定义
设函数 $f(x)$ 定义在区间 $I$ 上，若存在可导函数 $F(x)$，使得对任意 $x \in I$ 有：
$$ F'(x) = f(x) \quad \text{或} \quad dF(x) = f(x)dx $$
则称 $F(x)$ 为 $f(x)$ 在区间 $I$ 上的一个原函数。

### 2. 不定积分的定义
函数 $f(x)$ 的所有原函数的集合称为 $f(x)$ 的不定积分：
$$ \int f(x)dx = F(x) + C $$
其中 $C$ 为任意常数，称为积分常数。

### 3. 原函数存在定理
若 $f(x)$ 在区间 $I$ 上连续，则 $f(x)$ 在 $I$ 上必有原函数。

### 4. 不定积分的几何意义
$F(x) + C$ 表示一族曲线——积分曲线族，它们在横坐标相同处的切线斜率相等（都是 $f(x)$）。

## 二、 不定积分的四条基本性质

| 性质 | 数学表达 | 记忆口诀 |
|:---|:---|:---|
| 微分在先 | $[\int f(x)dx]' = f(x)$ | 先积后微，原形毕露 |
| 积分在先 | $\int F'(x)dx = F(x) + C$ | 先微后积，尾巴加C |
| 数乘 | $\int kf(x)dx = k\int f(x)dx$ | 系数提到外面去 |
| 加减 | $\int[f \pm g]dx = \int f dx \pm \int g dx$ | 加减分开算 |

## 三、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：不定积分结果**必须包含常数 $C$**！考试中漏写 $C$ 直接扣分。
* 🚨 **易错雷区**：原函数存在定理只说"连续必有原函数"，但不连续也可能有原函数——不要反向否定。
* 🔑 **秒杀秘籍**：验证某个函数是否为原函数，直接对其求导——若导数等于被积函数，就是原函数！

## 四、 闭卷真题挑战

> **【真题演练】**：已知 $F(x)$ 是 $f(x)$ 的一个原函数，且 $F(x) = x\ln x$，求 $\int f(x)dx$。

> [!success]- 点击查看答案与解析
> **【正确答案】**：$\int f(x)dx = x\ln x + C$
>
> **【核心解析】**：
> 由原函数定义可知，若 $F(x)$ 是 $f(x)$ 的一个原函数，则不定积分 $\int f(x)dx = F(x) + C$。
> 此处 $F(x) = x\ln x$，所以 $\int f(x)dx = x\ln x + C$。
