# 3.5 第二类换元法之三角代换

> **学科系统**：高等数学 → 一元函数积分学 → 第二类换元法之三角代换
> **秒杀类比**：看到 $\sqrt{a^2 - x^2}$ 就像看到直角三角形——用三角函数的平方关系 $1-\sin^2 = \cos^2$ 消掉根号，把代数积分变成三角积分。

## 一、 核心知识解构

### 1. 三种标准三角代换

| 被积特征 | 代换公式 | 微分替换 |
|:---|:---|:---|
| $\sqrt{a^2 - x^2}$ | $x = a\sin t$ | $dx = a\cos t dt$ |
| $\sqrt{a^2 + x^2}$ | $x = a\tan t$ | $dx = a\sec^2 t dt$ |
| $\sqrt{x^2 - a^2}$ | $x = a\sec t$ | $dx = a\sec t\tan t dt$ |

### 2. 三角恒等式依据
- $1 - \sin^2 t = \cos^2 t$ $\to$ 去掉 $\sqrt{a^2 - x^2}$ 的根号
- $1 + \tan^2 t = \sec^2 t$ $\to$ 去掉 $\sqrt{a^2 + x^2}$ 的根号
- $\sec^2 t - 1 = \tan^2 t$ $\to$ 去掉 $\sqrt{x^2 - a^2}$ 的根号

### 3. 操作步骤
1. 判断特征，选择对应的三角代换
2. 换元、替换 $dx$、化简根式
3. 对三角函数积分
4. 用辅助直角三角形回代到 $x$

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：回代时要用**辅助直角三角形**确定三角函数值！
* 🔑 **秒杀秘籍**：记忆辅助三角形的画法——以 $x = a\sin t$ 为例，$\sin t = \frac{x}{a}$，画直角三角形，对边 $x$、斜边 $a$，邻边 $\sqrt{a^2-x^2}$。

## 三、 闭卷真题挑战

> **【真题演练】**：求 $\int \frac{dx}{\sqrt{4-x^2}}$。

> [!success]- 点击查看答案与解析
> **【正确答案】**：$\arcsin\frac{x}{2} + C$
>
> **【核心解析】**：
> 特征 $\sqrt{4 - x^2} = \sqrt{2^2 - x^2}$，令 $x = 2\sin t$。
> $dx = 2\cos t dt$，$\sqrt{4-x^2} = \sqrt{4-4\sin^2 t} = 2|\cos t|$。
> 在 $t \in (-\frac{\pi}{2},\frac{\pi}{2})$ 内 $\cos t > 0$，所以：
> $\int \frac{dx}{\sqrt{4-x^2}} = \int \frac{2\cos t dt}{2\cos t} = \int dt = t + C = \arcsin\frac{x}{2} + C$
