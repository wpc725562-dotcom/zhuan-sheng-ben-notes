# 3.3 第一类换元法（凑微分法）

> **学科系统**：高等数学 → 一元函数积分学 → 第一类换元法
> **秒杀类比**：凑微分就像"换衣服"——你把函数的一部分"穿成"某个中间变量的微分形式，然后整个积分变得简单明了。

## 一、 核心知识解构

### 1. 定理（第一类换元法）
设 $u = \varphi(x)$ 可导，且 $\int f(u)du = F(u) + C$，则：
$$ \int f[\varphi(x)]\varphi'(x)dx = F[\varphi(x)] + C $$

### 2. 操作步骤
1. **凑微分**：将被积表达式中的某部分凑成 $d\varphi(x)$
2. **令 $u = \varphi(x)$**：换元简化
3. **对 $u$ 积分**：套用基本积分公式
4. **回代 $u = \varphi(x)$**：得到关于 $x$ 的结果

### 3. 常见凑微分模式

| 凑微分形式 | $d\varphi(x)$ | 适用场景 |
|:---|:---|:---|
| $f(ax+b)$ | $d(ax+b) = adx$ | 线性内层 |
| $f(x^\alpha)x^{\alpha-1}$ | $d(x^\alpha) = \alpha x^{\alpha-1}dx$ | 幂函数内层 |
| $f(e^x)e^x$ | $d(e^x) = e^x dx$ | 指数内层 |
| $f(\ln x)\frac{1}{x}$ | $d(\ln x) = \frac{1}{x}dx$ | 对数内层 |
| $f(\sin x)\cos x$ | $d(\sin x) = \cos x dx$ | 三角内层 |

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：凑微分时系数要准确配平！例如 $\int \cos 2x dx$，$d(2x) = 2dx$，所以要多乘 $\frac{1}{2}$。
* 🔑 **秒杀秘籍**：看到 $\int \frac{1}{x\ln x}dx$，直接凑 $d(\ln x) = \frac{1}{x}dx$，秒杀为 $\ln|\ln x| + C$。

## 三、 闭卷真题挑战

> **【真题演练】**：求 $\int \frac{e^x}{1+e^x}dx$。

> [!success]- 点击查看答案与解析
> **【正确答案】**：$\ln(1+e^x) + C$
>
> **【核心解析】**：
> 观察分子是分母的导数（或近似）：$(1+e^x)' = e^x$。
> 凑微分：$d(1+e^x) = e^x dx$
> 所以：$\int \frac{e^x}{1+e^x}dx = \int \frac{d(1+e^x)}{1+e^x} = \ln|1+e^x| + C = \ln(1+e^x) + C$
