# 6.4 曲线积分

> **学科系统**：高等数学 → 重积分与曲线积分 → 曲线积分
> **秒杀类比**：对弧长的曲线积分是"算弯曲铁丝的质量"——沿曲线逐点累加。对坐标的曲线积分是"算沿着曲线走的功"——力沿路径拉物体做的功。一个是"标量沿曲线累加"，一个是"向量沿曲线投影累加"。

## 一、 核心知识解构

### 1. 对弧长的曲线积分（第一类曲线积分）

#### 定义
$$\int_L f(x,y) ds = \lim_{\lambda \to 0} \sum_{i=1}^n f(\xi_i, \eta_i) \Delta s_i$$

其中 $ds$ 是弧长微元。

#### 物理意义
密度为 $f(x,y)$ 的曲线 $L$ 的质量。

#### 计算方法——参数化
设曲线 $L: \begin{cases} x = \varphi(t) \\ y = \psi(t) \end{cases}$，$t \in [\alpha, \beta]$

$$ds = \sqrt{[\varphi'(t)]^2 + [\psi'(t)]^2} \, dt$$

$$\int_L f(x,y) ds = \int_\alpha^\beta f(\varphi(t), \psi(t)) \cdot \sqrt{[\varphi'(t)]^2 + [\psi'(t)]^2} \, dt$$

#### 特殊情形
| 曲线形式 | $ds$ | 公式 |
|:---|:---|:---|
| $y = y(x)$，$a \leq x \leq b$ | $ds = \sqrt{1 + (y')^2}\, dx$ | $\int_a^b f(x,y(x))\sqrt{1+(y')^2}dx$ |
| $x = x(y)$，$c \leq y \leq d$ | $ds = \sqrt{1 + (x')^2}\, dy$ | $\int_c^d f(x(y),y)\sqrt{1+(x')^2}dy$ |
| 极坐标 $r = r(\theta)$ | $ds = \sqrt{r^2 + (r')^2}\, d\theta$ | $\int_{\theta_1}^{\theta_2} f(r\cos\theta, r\sin\theta)\sqrt{r^2+(r')^2}d\theta$ |

### 2. 对坐标的曲线积分（第二类曲线积分）

#### 定义
$$\int_L P(x,y) dx + Q(x,y) dy = \lim_{\lambda \to 0} \sum_{i=1}^n [P(\xi_i,\eta_i)\Delta x_i + Q(\xi_i,\eta_i)\Delta y_i]$$

#### 物理意义
变力 $\vec{F} = (P,Q)$ 沿曲线 $L$ 做的功。

#### 计算方法——参数化
设 $L: \begin{cases} x = \varphi(t) \\ y = \psi(t) \end{cases}$，$t$ 从 $\alpha$ 到 $\beta$（起点→终点）

$$\int_L Pdx + Qdy = \int_\alpha^\beta [P(\varphi(t),\psi(t))\varphi'(t) + Q(\varphi(t),\psi(t))\psi'(t)] dt$$

### 3. 两类曲线积分的对比

| 对比项 | 第一类（对弧长） | 第二类（对坐标） |
|:---|:---|:---|
| **物理意义** | 曲线质量 | 变力做功 |
| **积分变量** | $ds$（弧长） | $dx, dy$（坐标差） |
| **方向性** | 与方向无关 | 与方向有关（反向变号） |
| **计算要素** | $\sqrt{\varphi'^2 + \psi'^2}$ | $\varphi'(t), \psi'(t)$ |

### 4. 格林公式

沟通二重积分与曲线积分的桥梁：

$$\iint_D \left( \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) dxdy = \oint_L Pdx + Qdy$$

其中 $L$ 是 $D$ 的正向边界（逆时针方向）。

**用途**：将封闭曲线的曲线积分转化为二重积分，或将二重积分转化为曲线积分。

### 5. 曲线积分与路径无关的条件

若在单连通区域 $D$ 内 $\frac{\partial Q}{\partial x} = \frac{\partial P}{\partial y}$ 恒成立，则：
- $\int_L Pdx + Qdy$ 与路径无关（只与起点、终点有关）
- 存在函数 $u(x,y)$ 使 $du = Pdx + Qdy$（$Pdx + Qdy$ 是某函数的全微分）

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：第二类曲线积分**与方向有关**——方向相反时积分值变号，计算时一定要明确曲线的起点和终点
* 🚨 **易错雷区**：格林公式中 $L$ 必须是**封闭曲线**且取**正向**（逆时针）。如果是顺时针，结果要加负号
* 🚨 **易错雷区**：第一类曲线积分的 $ds$ 是弧长微元 $\sqrt{\varphi'^2 + \psi'^2}dt$，不要跟第二类的 $\varphi'(t)dt$ 混淆
* 🔑 **秒杀秘籍**：封闭曲线上的曲线积分 → 优先考虑格林公式
* 🔑 **秒杀秘籍**：验证 $\frac{\partial Q}{\partial x} = \frac{\partial P}{\partial y}$ → 如果成立，积分与路径无关，可沿折线计算
* 🔑 **秒杀秘籍**：第一类曲线积分用对称性——若 $L$ 关于 $x$ 轴对称，$f$ 对 $y$ 奇函数则积分值为 0

## 三、 闭卷真题挑战

> **【真题演练】**：计算 $\int_L xy\, ds$，其中 $L$ 是抛物线 $y = x^2$ 上从 $(0,0)$ 到 $(1,1)$ 的一段弧。

> [!success]- 点击查看答案与解析
> **【正确答案】**：
> $y = x^2$，$y' = 2x$，$x$ 从 $0$ 到 $1$
> $$ds = \sqrt{1 + (y')^2}\, dx = \sqrt{1 + 4x^2}\, dx$$
>
> $$\int_L xy\, ds = \int_0^1 x \cdot x^2 \cdot \sqrt{1 + 4x^2}\, dx = \int_0^1 x^3 \sqrt{1 + 4x^2}\, dx$$
>
> 令 $u = 1 + 4x^2$，则 $du = 8x\, dx$，$x^2 = \frac{u-1}{4}$，$x^3 dx = x^2 \cdot x dx = \frac{u-1}{4} \cdot \frac{du}{8} = \frac{u-1}{32} du$
>
> $x=0 \to u=1$，$x=1 \to u=5$
>
> 原式 $= \int_1^5 \frac{u-1}{32} \cdot \sqrt{u}\, du = \frac{1}{32} \int_1^5 (u^{3/2} - u^{1/2}) du$
>
> $= \frac{1}{32} \left[ \frac{2}{5}u^{5/2} - \frac{2}{3}u^{3/2} \right]_1^5$
>
> $= \frac{1}{32} \left[ \frac{2}{5}(5^{5/2} - 1) - \frac{2}{3}(5^{3/2} - 1) \right]$
>
> $= \frac{1}{48} \left[ \frac{6}{5}(25\sqrt{5} - 1) - 2(5\sqrt{5} - 1) \right]$
>
> **【核心解析】**：
> 第一类曲线积分的基本计算：曲线方程代入 $ds$，变量替换后化为一元定积分。注意 $ds = \sqrt{1+(y')^2}dx$ 这个公式。

> **【真题演练】**：利用格林公式计算 $\oint_L (xy + y^2)dx + (x^2 - xy)dy$，其中 $L$ 是顶点 $(0,0)$、$(1,0)$、$(1,1)$ 的三角形正向边界。

> [!success]- 点击查看答案与解析
> **【正确答案】**：
> $P = xy + y^2$，$Q = x^2 - xy$
>
> $\frac{\partial Q}{\partial x} = 2x - y$，$\frac{\partial P}{\partial y} = x + 2y$
>
> 由格林公式：
> $$\oint_L Pdx + Qdy = \iint_D \left( \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) dxdy$$
> $$= \iint_D [(2x - y) - (x + 2y)] dxdy = \iint_D (x - 3y) dxdy$$
>
> 区域 $D$：$0 \leq x \leq 1$，$0 \leq y \leq x$
> $$= \int_0^1 \int_0^x (x - 3y) dy\, dx = \int_0^1 \left[ xy - \frac{3}{2}y^2 \right]_0^x dx$$
> $$= \int_0^1 \left( x^2 - \frac{3}{2}x^2 \right) dx = \int_0^1 \left( -\frac{1}{2}x^2 \right) dx = -\frac{1}{6}$$
>
> **【核心解析】**：
> 格林公式标准用法：封闭曲线 $\to$ 求偏导 $\to$ 二重积分。注意 $L$ 是正向（逆时针），三角形区域边界上的曲线积分转化为三角形内部的二重积分。
