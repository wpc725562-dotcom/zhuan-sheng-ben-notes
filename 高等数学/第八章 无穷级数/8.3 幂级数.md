# 8.3 幂级数

> **学科系统**：高等数学 → 无穷级数 → 幂级数
> **秒杀类比**：幂级数就是用"多项式"逼近各种函数——就像用乐高积木拼出任意形状。$e^x$、$\sin x$、$\cos x$ 这些函数都可以展开成幂级数，每一项都是 $x^n$ 的倍数。收敛半径告诉你这个"乐高模型"在什么范围内是有效的。

## 一、 核心知识解构

### 1. 幂级数的定义

$$\sum_{n=0}^\infty a_n x^n = a_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n + \cdots$$

更一般的形式：$\sum_{n=0}^\infty a_n (x - x_0)^n$

### 2. 收敛半径与收敛域

#### 收敛半径的计算

$$R = \lim_{n\to\infty} \left| \frac{a_n}{a_{n+1}} \right| \quad \text{或} \quad R = \frac{1}{\limsup_{n\to\infty} \sqrt[n]{|a_n|}}$$

#### 收敛区间
- $|x| < R$：**绝对收敛**
- $|x| > R$：**发散**
- $|x| = R$（端点）：需**单独判定**

### 3. 幂级数的四则运算

在收敛区间内：
- 加减法：$\sum a_n x^n \pm \sum b_n x^n = \sum (a_n \pm b_n) x^n$
- 乘法：$(\sum a_n x^n)(\sum b_n x^n) = \sum c_n x^n$（$c_n = \sum_{k=0}^n a_k b_{n-k}$）

### 4. 幂级数的分析性质

设 $S(x) = \sum_{n=0}^\infty a_n x^n$，收敛半径为 $R$：

| 性质 | 表达式 |
|:---|:---|
| **逐项可积** | $\int_0^x S(t) dt = \sum_{n=0}^\infty \frac{a_n}{n+1} x^{n+1}$，$|x| < R$ |
| **逐项可导** | $S'(x) = \sum_{n=1}^\infty n a_n x^{n-1}$，$|x| < R$ |

> 逐项求导和逐项积分不改变收敛半径！

### 5. 泰勒级数 / 幂级数展开

$$f(x) = \sum_{n=0}^\infty \frac{f^{(n)}(x_0)}{n!} (x - x_0)^n$$

#### 常用麦克劳林展开（$x_0 = 0$）

| 函数 | 展开式 | 成立范围 |
|:---|:---|:---:|
| $\frac{1}{1-x}$ | $\sum_{n=0}^\infty x^n$ | $\|x\| < 1$ |
| $\frac{1}{1+x}$ | $\sum_{n=0}^\infty (-1)^n x^n$ | $\|x\| < 1$ |
| $e^x$ | $\sum_{n=0}^\infty \frac{x^n}{n!}$ | $(-\infty, +\infty)$ |
| $\sin x$ | $\sum_{n=0}^\infty \frac{(-1)^n x^{2n+1}}{(2n+1)!}$ | $(-\infty, +\infty)$ |
| $\cos x$ | $\sum_{n=0}^\infty \frac{(-1)^n x^{2n}}{(2n)!}$ | $(-\infty, +\infty)$ |
| $\ln(1+x)$ | $\sum_{n=1}^\infty \frac{(-1)^{n-1} x^n}{n}$ | $(-1, 1]$ |
| $(1+x)^\alpha$ | $1 + \sum_{n=1}^\infty \frac{\alpha(\alpha-1)\cdots(\alpha-n+1)}{n!} x^n$ | $\|x\| < 1$ |

### 6. 函数展开为幂级数的方法

1. **直接法**：计算 $f^{(n)}(0)$，代入泰勒公式
2. **间接法**：利用已知展开式，通过变量替换、四则运算、逐项求导/积分得到

> **推荐**：间接法比直接法简单得多，优先使用！

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：收敛半径公式是 $R = \lim |\frac{a_n}{a_{n+1}}|$，不是 $\lim |\frac{a_{n+1}}{a_n}|$——不要搞反了！
* 🚨 **易错雷区**：端点处（$x = \pm R$）必须**单独判断**——不能直接从收敛半径得出端点敛散性
* 🚨 **易错雷区**：逐项求导和逐项积分不改变收敛半径，但可能改变端点处的敛散性
* 🔑 **秒杀秘籍**：看到 $\frac{1}{ax+b}$ 形式 → 凑成 $\frac{1}{1-u}$ 形式展开
* 🔑 **秒杀秘籍**：展开函数时优先用间接法——从已知的 5 个基本展开式出发
* 🔑 **秒杀秘籍**：逐项积分常用于 $\ln(1+x)$ 的展开，逐项求导常用于 $\frac{1}{(1-x)^2}$ 的展开

## 三、 闭卷真题挑战

> **【真题演练】**：求幂级数 $\sum_{n=1}^\infty \frac{x^n}{n}$ 的收敛半径和收敛域。

> [!success]- 点击查看答案与解析
> **【正确答案】**：
> $a_n = \frac{1}{n}$
>
> $$R = \lim_{n\to\infty} \left| \frac{a_n}{a_{n+1}} \right| = \lim_{n\to\infty} \frac{1/n}{1/(n+1)} = \lim_{n\to\infty} \frac{n+1}{n} = 1$$
>
> 收敛半径 $R = 1$。考察端点：
>
> 当 $x = 1$ 时：$\sum \frac{1}{n}$——调和级数，**发散**
> 当 $x = -1$ 时：$\sum \frac{(-1)^n}{n}$——交错调和级数，**收敛**
>
> 收敛域为 $[-1, 1)$。
>
> **【核心解析】**：
> 先求收敛半径（比值法），再单独判断两个端点。注意收敛域可能有开有闭，不是对称的。

> **【真题演练】**：将函数 $f(x) = \frac{1}{1-2x}$ 展开为 $x$ 的幂级数。

> [!success]- 点击查看答案与解析
> **【正确答案】**：
> 利用已知展开式 $\frac{1}{1-u} = \sum_{n=0}^\infty u^n$，$|u| < 1$
>
> 令 $u = 2x$：
> $$\frac{1}{1-2x} = \sum_{n=0}^\infty (2x)^n = \sum_{n=0}^\infty 2^n x^n$$
>
> $|2x| < 1 \implies |x| < \frac{1}{2}$
>
> 所以展开式为 $\sum_{n=0}^\infty 2^n x^n$，收敛范围为 $|x| < \frac{1}{2}$。
>
> **【核心解析】**：
> 间接法展开——从已知的 $\frac{1}{1-u}$ 展开式出发，通过变量替换得到。不需要求导。
