# 8.4 傅里叶级数

> **学科系统**：高等数学 → 无穷级数 → 傅里叶级数
> **秒杀类比**：傅里叶级数就像"用音叉合成音乐"——任何一个周期声音（方波、锯齿波等）都可以分解成不同频率的正弦波和余弦波的叠加。在不同频率上分配不同音量（系数），就能"拼出"任意周期函数。

## 一、 核心知识解构

### 1. 周期函数的傅里叶级数

周期为 $2\pi$ 的函数 $f(x)$：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^\infty (a_n \cos nx + b_n \sin nx)$$

#### 傅里叶系数

$$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \cos nx \, dx, \quad n = 0,1,2,\cdots$$

$$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \sin nx \, dx, \quad n = 1,2,3,\cdots$$

### 2. 周期为 $2l$ 的傅里叶级数

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^\infty \left( a_n \cos\frac{n\pi x}{l} + b_n \sin\frac{n\pi x}{l} \right)$$

$$a_n = \frac{1}{l} \int_{-l}^{l} f(x) \cos\frac{n\pi x}{l} dx, \quad b_n = \frac{1}{l} \int_{-l}^{l} f(x) \sin\frac{n\pi x}{l} dx$$

### 3. 收敛定理（狄利克雷充分条件）

若 $f(x)$ 在一个周期内：
1. 连续或只有有限个第一类间断点
2. 只有有限个极值点

则傅里叶级数收敛，且：
- 在连续点：收敛到 $f(x)$
- 在间断点：收敛到 $\frac{f(x^-) + f(x^+)}{2}$

### 4. 奇延拓与偶延拓

#### 正弦级数（奇延拓）
将 $[0, l]$ 上的函数奇延拓到 $[-l, l]$：

$$a_n = 0, \quad b_n = \frac{2}{l} \int_0^l f(x) \sin\frac{n\pi x}{l} dx$$

$$f(x) \sim \sum_{n=1}^\infty b_n \sin\frac{n\pi x}{l}$$

#### 余弦级数（偶延拓）
将 $[0, l]$ 上的函数偶延拓到 $[-l, l]$：

$$a_0 = \frac{2}{l} \int_0^l f(x) dx, \quad a_n = \frac{2}{l} \int_0^l f(x) \cos\frac{n\pi x}{l} dx, \quad b_n = 0$$

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^\infty a_n \cos\frac{n\pi x}{l}$$

### 5. 傅里叶级数的物理意义

- **基波**：$n=1$ 的项（频率最低）
- **谐波**：$n \geq 2$ 的项（高次谐波）
- **频谱**：$a_n$ 和 $b_n$ 的分布描述了信号的频率成分

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：傅里叶级数在**间断点处收敛到左右极限的平均值**，不是函数值本身
* 🚨 **易错雷区**：$a_0$ 的公式分母是 $l$（或 $\pi$），而且级数中 $a_0$ 要除以 $2$——这两个地方容易出错
* 🚨 **易错雷区**：奇延拓后得到的是正弦级数，偶延拓得到的是余弦级数——不要搞混
* 🔑 **秒杀秘籍**：计算傅里叶系数时，先判断 $f(x)$ 的奇偶性——奇函数 $a_n = 0$，偶函数 $b_n = 0$，可省去一半计算
* 🔑 **秒杀秘籍**：方波函数的傅里叶级数只含正弦项（奇函数），三角波只含余弦项（偶函数）
* 🔑 **秒杀秘籍**：在 $[0,l]$ 上展开时，可以根据需要选择正弦级数或余弦级数

## 三、 闭卷真题挑战

> **【真题演练】**：将函数 $f(x) = x$（$-\pi < x < \pi$）展开为傅里叶级数。

> [!success]- 点击查看答案与解析
> **【正确答案】**：
> $f(x) = x$ 是奇函数，在 $[-\pi, \pi]$ 上：
> $a_n = 0$（所有 $n$）
>
> $$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} x \sin nx \, dx = \frac{2}{\pi} \int_0^{\pi} x \sin nx \, dx$$
>
> $$= \frac{2}{\pi} \left[ -\frac{x\cos nx}{n} + \frac{\sin nx}{n^2} \right]_0^{\pi} = \frac{2}{\pi} \left( -\frac{\pi\cos n\pi}{n} \right) = -\frac{2\cos n\pi}{n} = \frac{2(-1)^{n+1}}{n}$$
>
> 所以：
> $$x \sim \sum_{n=1}^\infty \frac{2(-1)^{n+1}}{n} \sin nx, \quad x \in (-\pi, \pi)$$
>
> 在 $x = \pm\pi$ 处，级数收敛到 $\frac{\pi + (-\pi)}{2} = 0$（跳跃间断点）。
>
> **【核心解析】**：
> 奇函数 $x$ 的傅里叶级数只含正弦项，省去 $a_n$ 的计算。利用分部积分求 $b_n$，最后注意间断点处的收敛值。
