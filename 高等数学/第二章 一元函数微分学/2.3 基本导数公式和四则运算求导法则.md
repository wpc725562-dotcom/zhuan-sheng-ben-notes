# 2.3 基本导数公式和四则运算求导法则

> **学科系统**：高等数学 → 一元函数微分学 → 基本导数公式
> **秒杀类比**：导数公式就像九九乘法表——背得滚瓜烂熟，做题才能快到飞起！

## 一、 基本求导公式表

### 1. 常数与幂函数
| 原函数 | 导数 |
|:---|:---:|
| $C$（常数） | $0$ |
| $x^n$ | $n x^{n-1}$ |
| $\sqrt{x}$ | $\frac{1}{2\sqrt{x}}$ |
| $\frac{1}{x}$ | $-\frac{1}{x^2}$ |

### 2. 三角函数
| 原函数 | 导数 |
|:---|:---:|
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $\tan x$ | $\sec^2 x$ |
| $\cot x$ | $-\csc^2 x$ |
| $\sec x$ | $\sec x \tan x$ |
| $\csc x$ | $-\csc x \cot x$ |

### 3. 反三角函数
| 原函数 | 导数 |
|:---|:---:|
| $\arcsin x$ | $\frac{1}{\sqrt{1-x^2}}$ |
| $\arccos x$ | $-\frac{1}{\sqrt{1-x^2}}$ |
| $\arctan x$ | $\frac{1}{1+x^2}$ |
| $\operatorname{arccot} x$ | $-\frac{1}{1+x^2}$ |

### 4. 指数对数函数
| 原函数 | 导数 |
|:---|:---:|
| $e^x$ | $e^x$ |
| $a^x$ | $a^x \ln a$ |
| $\ln x$ | $\frac{1}{x}$ |
| $\log_a x$ | $\frac{1}{x \ln a}$ |

## 二、 四则运算求导法则

设 $u = u(x), v = v(x)$ 均可导，则：

1. **加减法**：$(u \pm v)' = u' \pm v'$
2. **乘法**：$(uv)' = u'v + uv'$（前导后不导 + 前不导后导）
3. **数乘**：$(Cu)' = Cu'$
4. **除法**：$\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$（$v \neq 0$）

> **记忆口诀**：乘法口诀——前导后不导加前不导后导；除法口诀——上导下不导减上不导下导，除以分母平方。

## 三、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：$\tan x$ 的导数是 $\sec^2 x$，不是 $\sec x$！$\ln x$ 的导数是 $\frac{1}{x}$，不是 $\frac{1}{|x|}$！
* 🔑 **秒杀秘籍**：遇到复杂分式求导，比如 $f(x) = \frac{x^2+1}{x^2-1}$，先化简为 $f(x) = 1 + \frac{2}{x^2-1}$ 再求导可以省一半计算量！

## 四、 闭卷真题挑战

> **【真题演练】**：求函数 $f(x) = x^2 \ln x$ 的导数。

> [!success]- 点击查看答案与解析
> **【正确答案】**：$f'(x) = 2x \ln x + x$
>
> **【核心解析】**：
> 使用乘法法则（前导后不导 + 前不导后导）：
> $$ f'(x) = (x^2)' \ln x + x^2 (\ln x)' = 2x \cdot \ln x + x^2 \cdot \frac{1}{x} = 2x \ln x + x $$
