# 7.3 高阶线性微分方程

> **学科系统**：高等数学 → 常微分方程 → 高阶线性微分方程
> **秒杀类比**：高阶线性微分方程就像解密码——先解"特征方程"这个密码锁（代数方程），钥匙就是特征根。不同特征根对应不同的解形式，就像不同的密码对应不同的解锁结果。

## 一、 核心知识解构

### 1. 二阶线性微分方程的结构

$$y'' + P(x)y' + Q(x)y = f(x)$$

- 齐次：$y'' + Py' + Qy = 0$（$f(x) = 0$）
- 非齐次：$y'' + Py' + Qy = f(x)$（$f(x) \neq 0$）

**解的结构定理**：

| 情形 | 通解结构 |
|:---|:---|
| **齐次** | $y = C_1 y_1 + C_2 y_2$（$y_1, y_2$ 为两个线性无关的特解）|
| **非齐次** | $y = \bar{y} + y^*$（$\bar{y}$ 是对应齐次的通解，$y^*$ 是一个特解）|

### 2. 常系数齐次线性方程的解法

$$y'' + py' + qy = 0 \quad (p,q \text{ 为常数})$$

**特征方程**：$r^2 + pr + q = 0$

| 特征根情况 | 通解形式 |
|:---|:---|
| 两个不等实根 $r_1 \neq r_2$ | $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| 重根 $r_1 = r_2 = r$ | $y = (C_1 + C_2 x)e^{rx}$ |
| 共轭复根 $r_{1,2} = \alpha \pm i\beta$ | $y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$ |

#### 推广到 $n$ 阶
特征方程 $r^n + a_1 r^{n-1} + \cdots + a_n = 0$：
- 每个单实根 $r$ → $e^{rx}$
- 每个 $k$ 重实根 $r$ → $e^{rx}, xe^{rx}, \cdots, x^{k-1}e^{rx}$
- 每个单重复根 $\alpha \pm i\beta$ → $e^{\alpha x}\cos\beta x, e^{\alpha x}\sin\beta x$
- 每对 $k$ 重复根 → $e^{\alpha x}\cos\beta x, xe^{\alpha x}\cos\beta x, \cdots$

### 3. 常系数非齐次线性方程的解法

$$y'' + py' + qy = f(x)$$

**解法**：先求齐次通解 $\bar{y}$，再求非齐次特解 $y^*$。

#### 特解待定系数法——$f(x) = P_m(x)e^{\lambda x}$

$$y^* = x^k \cdot Q_m(x) \cdot e^{\lambda x}$$

其中 $Q_m(x)$ 是与 $P_m(x)$ 同次的多项式，$k$ 的取值：

| $\lambda$ 与特征根的关系 | $k$ |
|:---|:---:|
| $\lambda$ 不是特征根 | $k = 0$ |
| $\lambda$ 是单重特征根 | $k = 1$ |
| $\lambda$ 是二重特征根 | $k = 2$ |

#### $f(x) = e^{\alpha x}[P_l(x)\cos\beta x + P_n(x)\sin\beta x]$

$$y^* = x^k e^{\alpha x}[R_m(x)\cos\beta x + S_m(x)\sin\beta x]$$

其中 $m = \max(l,n)$，$k$ 由 $\alpha \pm i\beta$ 是否为特征根决定（是→1，否→0）。

### 4. 可降阶的高阶方程

| 类型 | 形式 | 降阶方法 |
|:---|:---|:---|
| 缺 $y$ 型 | $y'' = f(x, y')$ | 令 $p = y'$，$y'' = p'$ |
| 缺 $x$ 型 | $y'' = f(y, y')$ | 令 $p = y'$，$y'' = p\frac{dp}{dy}$ |

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：特征方程写错符号！$y'' + py' + qy = 0$ 对应 $r^2 + pr + q = 0$——$y''$ 对应 $r^2$，$y'$ 对应 $r$，$y$ 对应 $1$
* 🚨 **易错雷区**：非齐次特解中 $x^k$ 的 $k$ 要选对——$\lambda$ 是几重特征根，$k$ 就是几。不是特征根时 $k=0$（即不乘 $x$）
* 🚨 **易错雷区**：特解 $y^*$ 中的多项式系数要用待定系数法确定——不是随便写的
* 🔑 **秒杀秘籍**：记住通解结构：齐次通解 + 非齐次特解（$\bar{y} + y^*$），两者是相加关系
* 🔑 **秒杀秘籍**：特征方程 $r^2 + pr + q = 0$ 的判别式 $\Delta = p^2 - 4q$ 决定通解类型
* 🔑 **秒杀秘籍**：看到 $f(x) = e^{\lambda x} \cdot \text{多项式}$ → 待定系数法三步走：① $\lambda$ 是几重根？② 设 $y^* = x^k Q_m(x) e^{\lambda x}$ ③ 代入原方程求系数

## 三、 闭卷真题挑战

> **【真题演练】**：求微分方程 $y'' - 3y' + 2y = 0$ 的通解。

> [!success]- 点击查看答案与解析
> **【正确答案】**：
> 特征方程：$r^2 - 3r + 2 = 0$
> $(r-1)(r-2) = 0$，$r_1 = 1$，$r_2 = 2$
>
> 两个不等实根，通解为：
> $$y = C_1 e^{x} + C_2 e^{2x}$$
>
> **【核心解析】**：
> 常系数齐次方程的标准解法：写出特征方程 → 求特征根 → 对照表格写通解。

> **【真题演练】**：求微分方程 $y'' - 2y' + y = e^x$ 的通解。

> [!success]- 点击查看答案与解析
> **【正确答案】**：
> **① 齐次通解 $\bar{y}$**
> 特征方程：$r^2 - 2r + 1 = (r-1)^2 = 0$，$r = 1$（二重根）
> $\bar{y} = (C_1 + C_2 x)e^x$
>
> **② 非齐次特解 $y^*$**
> $f(x) = e^x$，$\lambda = 1$ 是二重特征根 → $k = 2$
> 设 $y^* = A x^2 e^x$
> $y^{*'} = A(2x + x^2)e^x$，$y^{*''} = A(2 + 4x + x^2)e^x$
>
> 代入原方程：
> $A(2 + 4x + x^2)e^x - 2A(2x + x^2)e^x + A x^2 e^x = e^x$
> $A(2 + 4x + x^2 - 4x - 2x^2 + x^2) = 1$
> $A \cdot 2 = 1 \implies A = \frac{1}{2}$
>
> $y^* = \frac{1}{2}x^2 e^x$
>
> **③ 非齐次通解**
> $$y = \bar{y} + y^* = (C_1 + C_2 x)e^x + \frac{1}{2}x^2 e^x$$
>
> **【核心解析】**：
> 非齐次方程的标准解法：先求齐次通解，再设特解代入求系数，最后相加。注意 $\lambda = 1$ 是二重根所以 $k=2$，设 $y^* = Ax^2 e^x$。
