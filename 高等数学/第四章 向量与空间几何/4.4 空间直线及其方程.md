# 4.4 空间直线及其方程

> **学科系统**：高等数学 → 向量与空间几何 → 空间直线及其方程
> **秒杀类比**：空间直线就像一束激光——由一个起点和一个方向决定。直线的四种方程形式就像同一个人的四种不同描述方式：身份证号（对称式）、简历（参数式）、合照（两点式）、投影描述（一般式）。

## 一、 核心知识解构

### 1. 直线的方向向量

与直线平行的非零向量 $\vec{s} = (m, n, p)$ 称为直线的**方向向量**。直线的方向由方向向量决定。

### 2. 直线的对称式方程（点向式）

过点 $M_0(x_0, y_0, z_0)$，方向向量 $\vec{s} = (m, n, p)$：

$$\frac{x - x_0}{m} = \frac{y - y_0}{n} = \frac{z - z_0}{p}$$

### 3. 直线的参数式方程

$$\begin{cases}
x = x_0 + m t \\
y = y_0 + n t \\
z = z_0 + p t
\end{cases}$$

其中 $t$ 为参数。

### 4. 直线的两点式方程

过两点 $M_1(x_1, y_1, z_1)$ 和 $M_2(x_2, y_2, z_2)$：

$$\frac{x - x_1}{x_2 - x_1} = \frac{y - y_1}{y_2 - y_1} = \frac{z - z_1}{z_2 - z_1}$$

### 5. 直线的一般式方程（两平面交线）

$$\begin{cases}
A_1x + B_1y + C_1z + D_1 = 0 \\
A_2x + B_2y + C_2z + D_2 = 0
\end{cases}$$

方向向量：
$$\vec{s} = (A_1, B_1, C_1) \times (A_2, B_2, C_2)$$

### 6. 线线关系

#### 两直线的夹角（锐角）
直线 $L_1$ 方向向量 $\vec{s}_1 = (m_1, n_1, p_1)$
直线 $L_2$ 方向向量 $\vec{s}_2 = (m_2, n_2, p_2)$

$$\cos\varphi = \frac{|m_1 m_2 + n_1 n_2 + p_1 p_2|}{\sqrt{m_1^2 + n_1^2 + p_1^2} \cdot \sqrt{m_2^2 + n_2^2 + p_2^2}}$$

#### 位置关系
| 关系 | 条件 |
|:---|:---|
| 平行 | $\frac{m_1}{m_2} = \frac{n_1}{n_2} = \frac{p_1}{p_2}$ |
| 垂直 | $m_1 m_2 + n_1 n_2 + p_1 p_2 = 0$ |
| 共面 | 两直线方向向量与连接向量的混合积为 0 |
| 异面 | 混合积不为 0 |

### 7. 线面关系

#### 直线与平面的夹角
直线方向向量 $\vec{s} = (m, n, p)$，平面法向量 $\vec{n} = (A, B, C)$：

$$\sin\varphi = \frac{|Am + Bn + Cp|}{\sqrt{A^2 + B^2 + C^2} \cdot \sqrt{m^2 + n^2 + p^2}}$$

#### 位置关系
| 关系 | 条件 |
|:---|:---|
| 直线在平面内 | $Am + Bn + Cp = 0$ 且直线上一点满足平面方程 |
| 直线平行于平面 | $Am + Bn + Cp = 0$ 但点不满足平面方程 |
| 直线垂直于平面 | $\frac{A}{m} = \frac{B}{n} = \frac{C}{p}$（方向向量 ∥ 法向量）|

### 8. 点到直线的距离

点 $M_0$ 到过点 $M_1$、方向 $\vec{s}$ 的直线的距离：
$$d = \frac{\|\overrightarrow{M_1M_0} \times \vec{s}\|}{\|\vec{s}\|}$$

### 9. 平面束方程

过直线 $\begin{cases} A_1x + B_1y + C_1z + D_1 = 0 \\ A_2x + B_2y + C_2z + D_2 = 0 \end{cases}$ 的平面束：

$$(A_1x + B_1y + C_1z + D_1) + \lambda(A_2x + B_2y + C_2z + D_2) = 0$$

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：对称式方程中分母（方向向量的分量）可以为 0——如果 $m = 0$，则方程写作 $\frac{x - x_0}{0} = \frac{y - y_0}{n} = \frac{z - z_0}{p}$，理解为 $x = x_0$
* 🚨 **易错雷区**：线面夹角用**正弦**，线线、面面夹角用**余弦**。别搞混了！
* 🚨 **易错雷区**：直线的一般式是两个平面方程的联立——化简为对称式时，方向向量是两个法向量的**叉积**
* 🔑 **秒杀秘籍**：求直线方程三步走：① 找方向向量 ② 找直线上一点 ③ 代入对称式或参数式
* 🔑 **秒杀秘籍**：直线过点且与平面垂直 → 方向向量直接取平面的法向量
* 🔑 **秒杀秘籍**：直线过点且与两直线都垂直 → 方向向量取两直线方向向量的叉积

## 三、 闭卷真题挑战

> **【真题演练】**：求过点 $(1,0,-2)$ 且与直线 $\frac{x-1}{3} = \frac{y+2}{-4} = \frac{z}{2}$ 平行的直线方程。

> [!success]- 点击查看答案与解析
> **【正确答案】**：
> 已知直线的方向向量 $\vec{s} = (3, -4, 2)$
> 所求直线与之平行，故取同方向向量 $\vec{s} = (3, -4, 2)$
> 代入点 $(1,0,-2)$ 得对称式方程：
> $$\frac{x-1}{3} = \frac{y}{-4} = \frac{z+2}{2}$$
>
> **【核心解析】**：
> 两直线平行 $\iff$ 方向向量成比例。直接取相同的方向向量，换一个点即可。

> **【真题演练】**：求直线 $\begin{cases} x + y + z + 1 = 0 \\ 2x - y + 3z + 4 = 0 \end{cases}$ 的对称式方程。

> [!success]- 点击查看答案与解析
> **【正确答案】**：
> 法向量 $\vec{n}_1 = (1,1,1)$，$\vec{n}_2 = (2,-1,3)$
>
> 方向向量 $\vec{s} = \vec{n}_1 \times \vec{n}_2 = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 1 & 1 & 1 \\ 2 & -1 & 3 \end{vmatrix}$
> $= \vec{i}(3+1) - \vec{j}(3-2) + \vec{k}(-1-2) = (4, -1, -3)$
>
> 求直线上一点：令 $z = 0$，解 $\begin{cases} x + y + 1 = 0 \\ 2x - y + 4 = 0 \end{cases}$
> 相加得 $3x + 5 = 0 \implies x = -\frac{5}{3}$，代入得 $y = \frac{2}{3}$
> 直线上一点 $(-\frac{5}{3}, \frac{2}{3}, 0)$
>
> 对称式方程：$\frac{x + \frac{5}{3}}{4} = \frac{y - \frac{2}{3}}{-1} = \frac{z}{-3}$
>
> **【核心解析】**：
> 一般式→对称式：① 方向向量=两法向量叉积 ② 令一个变量为0求直线上一点 ③ 代入对称式
