# 4.1 空间直角坐标系与向量的线性运算

> **学科系统**：高等数学 → 向量与空间几何 → 空间直角坐标系与向量
> **秒杀类比**：向量就是"带箭头有方向的线段"——向东走3步和向东北走4步，用向量加起来就是你的实际位移。就像游戏里的人物移动指令：(x, y, z)三个数值决定你在三维空间里的位置。

## 一、 核心知识解构

### 1. 空间直角坐标系

三条互相垂直的数轴（$x$轴、$y$轴、$z$轴）按右手规则构成空间直角坐标系。

| 要素 | 说明 |
|:---|:---|
| 坐标原点 | $O(0,0,0)$ |
| 坐标轴 | $x$轴、$y$轴、$z$轴 |
| 坐标面 | $xOy$面、$yOz$面、$zOx$面 |
| 卦限 | 三个坐标面将空间分为 8 个卦限 |
| 点坐标 | $M(x,y,z)$ —— $x$为横坐标，$y$为纵坐标，$z$为竖坐标 |

### 2. 向量的基本概念

| 概念 | 定义 |
|:---|:---|
| **向量** | 既有大小又有方向的量，记作 $\vec{a}$ 或 $\overrightarrow{AB}$ |
| **模** | 向量的大小，记作 $\|\vec{a}\|$ |
| **方向** | 由方向角 $\alpha,\beta,\gamma$ 表示（与三个坐标轴的夹角） |
| **零向量** | 模为 0 的向量，记作 $\vec{0}$ |
| **单位向量** | 模为 1 的向量，记作 $\vec{e}$ |
| **相等向量** | 大小相等、方向相同的向量 |
| **自由向量** | 只考虑大小和方向，不考虑起点位置 |

### 3. 向量的坐标表示

点 $A(a_x,a_y,a_z)$ 到 $B(b_x,b_y,b_z)$ 的向量：
$$\overrightarrow{AB} = (b_x - a_x,\; b_y - a_y,\; b_z - a_z)$$

向量 $\vec{a} = (a_x, a_y, a_z)$ 的模：
$$\|\vec{a}\| = \sqrt{a_x^2 + a_y^2 + a_z^2}$$

### 4. 方向角与方向余弦

方向角 $\alpha,\beta,\gamma$ 是向量与三个坐标轴正方向的夹角（$0 \leq \alpha,\beta,\gamma \leq \pi$）。

方向余弦：
$$\cos\alpha = \frac{a_x}{\|\vec{a}\|},\quad \cos\beta = \frac{a_y}{\|\vec{a}\|},\quad \cos\gamma = \frac{a_z}{\|\vec{a}\|}$$

恒等式：
$$\cos^2\alpha + \cos^2\beta + \cos^2\gamma = 1$$

### 5. 向量的线性运算

#### 加法（平行四边形法则 / 三角形法则）
$$\vec{a} + \vec{b} = (a_x + b_x,\; a_y + b_y,\; a_z + b_z)$$

#### 减法
$$\vec{a} - \vec{b} = (a_x - b_x,\; a_y - b_y,\; a_z - b_z)$$

#### 数乘
$$\lambda\vec{a} = (\lambda a_x,\; \lambda a_y,\; \lambda a_z)$$

| 运算律 | 表达式 |
|:---|:---|
| 交换律 | $\vec{a} + \vec{b} = \vec{b} + \vec{a}$ |
| 结合律 | $(\vec{a} + \vec{b}) + \vec{c} = \vec{a} + (\vec{b} + \vec{c})$ |
| 数乘分配律 | $\lambda(\vec{a} + \vec{b}) = \lambda\vec{a} + \lambda\vec{b}$ |
| 数乘结合律 | $\lambda(\mu\vec{a}) = (\lambda\mu)\vec{a}$ |

### 6. 向量的投影

向量 $\vec{a}$ 在向量 $\vec{b}$ 上的投影：
$$\text{Prj}_{\vec{b}}\vec{a} = \|\vec{a}\| \cos\theta = \frac{\vec{a} \cdot \vec{b}}{\|\vec{b}\|}$$

其中 $\theta$ 为 $\vec{a}$ 与 $\vec{b}$ 的夹角。

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：零向量的方向是任意的，不是"没有方向"——在某些证明题中这个细节很关键
* 🚨 **易错雷区**：向量的模是标量（非负实数），不是向量。$\|\vec{a}\|$ 是一个数！
* 🚨 **易错雷区**：空间点的坐标顺序是 $(x,y,z)$，不能写错顺序。$(1,2,3)$ 和 $(3,2,1)$ 是两个不同的点
* 🔑 **秒杀秘籍**：已知两点求向量 = 终点坐标减起点坐标（"终减起"）。记牢这个口诀，永远不会搞反方向
* 🔑 **秒杀秘籍**：判断三点共线 → 验证两个向量是否平行（即一个向量是另一个的倍数）
* 🔑 **秒杀秘籍**：单位向量 $\vec{e} = \frac{\vec{a}}{\|\vec{a}\|}$ ——任何非零向量除以自己的模就是同向单位向量

## 三、 闭卷真题挑战

> **【真题演练】**：已知两点 $A(4,0,5)$ 和 $B(7,1,3)$，求向量 $\overrightarrow{AB}$ 的模及方向余弦。

> [!success]- 点击查看答案与解析
> **【正确答案】**：
> $$\overrightarrow{AB} = (7-4,\; 1-0,\; 3-5) = (3,1,-2)$$
> $$\|\overrightarrow{AB}\| = \sqrt{3^2 + 1^2 + (-2)^2} = \sqrt{9+1+4} = \sqrt{14}$$
> $$\cos\alpha = \frac{3}{\sqrt{14}},\quad \cos\beta = \frac{1}{\sqrt{14}},\quad \cos\gamma = \frac{-2}{\sqrt{14}}$$
> 验证：$\frac{9}{14} + \frac{1}{14} + \frac{4}{14} = \frac{14}{14} = 1$ ✅
>
> **【核心解析】**：
> 1. 先求向量坐标：**终点减起点**（$B$ 减 $A$）
> 2. 再求模：三个分量平方和开根号
> 3. 方向余弦 = 每个分量除以模
> 4. 最后用 $\cos^2\alpha + \cos^2\beta + \cos^2\gamma = 1$ 验证结果

> **【真题演练】**：设向量 $\vec{a} = (1,2,-2)$，求与 $\vec{a}$ 同方向的单位向量。

> [!success]- 点击查看答案与解析
> **【正确答案】**：
> $$\|\vec{a}\| = \sqrt{1^2 + 2^2 + (-2)^2} = \sqrt{9} = 3$$
> $$\vec{e} = \frac{\vec{a}}{\|\vec{a}\|} = \left(\frac{1}{3},\; \frac{2}{3},\; -\frac{2}{3}\right)$$
>
> **【核心解析】**：
> 单位向量 = 原向量 ÷ 模。验证：$\left(\frac{1}{3}\right)^2 + \left(\frac{2}{3}\right)^2 + \left(-\frac{2}{3}\right)^2 = \frac{1+4+4}{9} = 1$ ✅
