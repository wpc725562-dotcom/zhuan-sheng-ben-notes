# 7.2 一阶微分方程

> **学科系统**：高等数学 → 常微分方程 → 一阶微分方程
> **秒杀类比**：一阶微分方程就像不同的"解锁方式"——可分离变量是"直接拆锁"（两边各自积分），齐次方程是"换个钥匙开"（变量替换），一阶线性是"按标准流程开"（套公式）。

## 一、 核心知识解构

### 1. 可分离变量的微分方程

**形式**：$\frac{dy}{dx} = g(x)h(y)$ 或 $M_1(x)N_1(y)dx + M_2(x)N_2(y)dy = 0$

**解法**：
$$\frac{dy}{dx} = g(x)h(y) \implies \frac{dy}{h(y)} = g(x)dx \implies \int \frac{dy}{h(y)} = \int g(x)dx$$

### 2. 齐次微分方程

**形式**：$\frac{dy}{dx} = f\left(\frac{y}{x}\right)$

**解法**：令 $u = \frac{y}{x}$，则 $y = ux$，$\frac{dy}{dx} = u + x\frac{du}{dx}$，代入后化为可分离变量。

### 3. 一阶线性微分方程

**标准形式**：$y' + P(x)y = Q(x)$

**通解公式**：
$$y = e^{-\int P(x)dx} \left[ \int Q(x) e^{\int P(x)dx} dx + C \right]$$

#### 推导记忆口诀
1. 先求齐次解：$y' + Py = 0 \implies y = Ce^{-\int P dx}$
2. 再用常数变易法：$C \to C(x)$，代入非齐次方程得 $C(x) = \int Q e^{\int P dx} dx + C$
3. 最终：$y = e^{-\int P dx}\left( \int Q e^{\int P dx} dx + C \right)$

#### 伯努利方程
**形式**：$y' + P(x)y = Q(x)y^n$（$n \neq 0, 1$）

**解法**：令 $z = y^{1-n}$，化为 $z' + (1-n)P(x)z = (1-n)Q(x)$，按一阶线性求解。

### 4. 全微分方程

**形式**：$M(x,y)dx + N(x,y)dy = 0$，且 $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$

**解法**：存在 $u(x,y)$ 使 $du = Mdx + Ndy$，通解为 $u(x,y) = C$。

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：一阶线性方程的标准形式是 $y' + P(x)y = Q(x)$——如果 $y'$ 的系数不是 1，先除以系数化为标准形式
* 🚨 **易错雷区**：套公式时，$\int P(x)dx$ 和 $\int Q(x)e^{\int P(x)dx}dx$ 都要带积分常数——但最终只需一个 $C$
* 🚨 **易错雷区**：分离变量时，分母为 0 的情况可能漏解——检查 $h(y) = 0$ 是否也是解
* 🔑 **秒杀秘籍**：看到 $\frac{y}{x}$ 形式的表达式 → 齐次方程，令 $u = \frac{y}{x}$
* 🔑 **秒杀秘籍**：一阶线性方程直接用公式，不要每次重新推导常数变易法
* 🔑 **秒杀秘籍**：记住 $e^{\int P dx}$ 这个积分因子——它是一阶线性方程的核心

## 三、 闭卷真题挑战

> **【真题演练】**：求微分方程 $\frac{dy}{dx} = \frac{y}{x} + x^2$ 的通解。

> **点击查看答案与解析**
> **【正确答案】**：
> 整理为标准形式：$y' - \frac{1}{x}y = x^2$
>
> $P(x) = -\frac{1}{x}$，$Q(x) = x^2$
>
> $\int P(x)dx = \int -\frac{1}{x}dx = -\ln|x|$
>
> $e^{\int P dx} = e^{-\ln|x|} = \frac{1}{|x|}$，常用 $\frac{1}{x}$（考虑 $x>0$）
>
> $y = e^{-\int P dx} \left[ \int Q e^{\int P dx} dx + C \right]$
>
> $= e^{\ln x} \left[ \int x^2 \cdot \frac{1}{x} dx + C \right]$
>
> $= x \left[ \int x\, dx + C \right] = x \left( \frac{x^2}{2} + C \right) = \frac{x^3}{2} + Cx$
>
> **【核心解析】**：
> 标准的一阶线性方程。先化标准形式，再套公式。注意 $e^{-\ln x} = \frac{1}{x}$，$e^{\ln x} = x$。

> **【真题演练】**：求微分方程 $y' - \frac{2}{x+1}y = (x+1)^3$ 的通解。

> **点击查看答案与解析**
> **【正确答案】**：
> $P(x) = -\frac{2}{x+1}$，$Q(x) = (x+1)^3$
>
> $\int P dx = \int -\frac{2}{x+1}dx = -2\ln|x+1| = \ln[(x+1)^{-2}]$
>
> $e^{\int P dx} = (x+1)^{-2} = \frac{1}{(x+1)^2}$
>
> $e^{-\int P dx} = (x+1)^2$
>
> $y = e^{-\int P dx} \left[ \int Q e^{\int P dx} dx + C \right]$
>
> $= (x+1)^2 \left[ \int (x+1)^3 \cdot \frac{1}{(x+1)^2} dx + C \right]$
>
> $= (x+1)^2 \left[ \int (x+1) dx + C \right]$
>
> $= (x+1)^2 \left[ \frac{(x+1)^2}{2} + C \right]$
>
> $= \frac{1}{2}(x+1)^4 + C(x+1)^2$
>
> **【核心解析】**：
> 同样是套公式的典型题。关键是正确计算 $e^{\int P dx}$ 和 $e^{-\int P dx}$，然后代入公式。公式中的积分顺序不要搞反。
