# 3.4 第二类换元法之根式代换

> **学科系统**：高等数学 → 一元函数积分学 → 第二类换元法之根式代换
> **秒杀类比**：根式代换就像"脱掉根号外套"——令整个根式为 $t$，解出 $x$ 关于 $t$ 的表达式，从而去掉根号，转化为有理函数积分。

## 一、 核心知识解构

### 1. 根式代换的适用场景
被积函数中含有形如 $\sqrt[n]{ax+b}$ 或 $\sqrt{\frac{ax+b}{cx+d}}$ 的根式。

### 2. 代换方法
令 $t = \sqrt[n]{ax+b}$，则 $x = \frac{t^n - b}{a}$，$dx = \frac{n}{a}t^{n-1}dt$。

### 3. 操作步骤
1. **令根式为 $t$**：设 $t = \sqrt[n]{ax+b}$
2. **反解 $x$**：$x = \frac{t^n - b}{a}$
3. **求 $dx$**：$dx = \frac{n}{a}t^{n-1}dt$
4. **代入化简**：原积分化为关于 $t$ 的有理函数积分
5. **回代**：将 $t = \sqrt[n]{ax+b}$ 代回

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：第二类换元后，$dx$ 也要跟着换！不要只换 $x$ 不换 $dx$。
* 🚨 **易错雷区**：回代时别忘了把积分结果中的 $t$ 全部换成 $x$ 的表达式。
* 🔑 **秒杀秘籍**：含多个不同根次根式时，取最小公倍数为 $n$。如含 $\sqrt{x}$ 和 $\sqrt[3]{x}$，令 $t = \sqrt[6]{x}$（6 是 2 和 3 的最小公倍数）。

## 三、 闭卷真题挑战

> **【真题演练】**：求 $\int \frac{dx}{1+\sqrt{x}}$。

> **点击查看答案与解析**
> **【正确答案】**：$2\sqrt{x} - 2\ln(1+\sqrt{x}) + C$
>
> **【核心解析】**：
> 令 $t = \sqrt{x}$，则 $x = t^2$，$dx = 2tdt$。
> 代入：$\int \frac{dx}{1+\sqrt{x}} = \int \frac{2tdt}{1+t} = 2\int \frac{t}{1+t}dt = 2\int(1-\frac{1}{1+t})dt$
> $= 2(t - \ln|1+t|) + C = 2\sqrt{x} - 2\ln(1+\sqrt{x}) + C$
