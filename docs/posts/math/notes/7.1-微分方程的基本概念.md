# 7.1 微分方程的基本概念

> **学科系统**：高等数学 → 常微分方程 → 基本概念
> **秒杀类比**：微分方程就是"已知导数找原函数"的升级版——就像你只知道速度的变化规律，要反推出位置随时间的变化。微分方程描述的是"变化率之间的关系"，解方程就是找到满足这种关系的函数。

## 一、 核心知识解构

### 1. 微分方程的定义

含有未知函数及其导数的方程称为**微分方程**。

- **常微分方程**：未知函数是一元函数
- **偏微分方程**：未知函数是多元函数

### 2. 微分方程的阶

方程中出现的最高阶导数的阶数称为微分方程的**阶**。

$$y''' + 2y'' - y' + 3y = 0 \quad (\text{三阶})$$
$$y' = 2xy \quad (\text{一阶})$$

### 3. 微分方程的解

| 类型 | 定义 | 例子（对 $y' = 2x$） |
|:---|:---|:---:|
| **通解** | 含有任意常数的解，常数个数等于阶数 | $y = x^2 + C$ |
| **特解** | 通解中常数取特定值 | $y = x^2 + 1$ |
| **初始条件** | 确定特解的条件（定常数用） | $y(0) = 1$ |

### 4. 初值问题

微分方程 + 初始条件 = **初值问题（柯西问题）**

$$\begin{cases}
y' = f(x,y) \\
y(x_0) = y_0
\end{cases}$$

### 5. 几何意义

微分方程 $\frac{dy}{dx} = f(x,y)$ 在每点 $(x,y)$ 给出了切线的斜率——称为**方向场**。解曲线是方向场中处处与方向相切的曲线。

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：通解中的任意常数个数 = 微分方程的阶数。一阶一个 $C$，二阶两个 $C_1, C_2$
* 🚨 **易错雷区**：有些方程的解不能写成显式 $y = f(x)$，只能写成隐式 $F(x,y) = 0$——这称为隐式解
* 🔑 **秒杀秘籍**：验证某函数是否为解——直接代入微分方程看是否成立
* 🔑 **秒杀秘籍**：确定通解中常数的值——利用初始条件代入解方程

## 三、 闭卷真题挑战

> **【真题演练】**：验证 $y = e^{2x}$ 是微分方程 $y'' - 3y' + 2y = 0$ 的解。

> **点击查看答案与解析**
> **【正确答案】**：
> $y = e^{2x}$，$y' = 2e^{2x}$，$y'' = 4e^{2x}$
>
> 代入方程：
> $y'' - 3y' + 2y = 4e^{2x} - 3(2e^{2x}) + 2(e^{2x})$
> $= 4e^{2x} - 6e^{2x} + 2e^{2x} = 0$ ✅
>
> 所以 $y = e^{2x}$ 是方程的解。
>
> **【核心解析】**：
> 验证解的标准方法：求导 → 代入 → 看等式是否成立。
