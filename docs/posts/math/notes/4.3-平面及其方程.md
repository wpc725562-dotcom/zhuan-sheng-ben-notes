# 4.3 平面及其方程

> **学科系统**：高等数学 → 向量与空间几何 → 平面及其方程
> **秒杀类比**：确定一个平面就像用三脚架固定相机——一个点 + 一个方向（法向量）就能锁定一个平面。或者说：平面是所有与法向量垂直的点的集合。

## 一、 核心知识解构

### 1. 平面的点法式方程

已知平面过点 $M_0(x_0, y_0, z_0)$，法向量 $\vec{n} = (A, B, C)$：

$$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$$

**法向量** $\vec{n}$：与平面垂直的向量，是平面方程的灵魂。

### 2. 平面的一般式方程

$$Ax + By + Cz + D = 0$$

其中 $\vec{n} = (A, B, C)$ 为法向量。

#### 特殊情况
| 方程特点 | 平面位置 |
|:---|:---|
| $D = 0$ | 过原点 |
| 缺 $x$ 项（$A = 0$） | 平行于 $x$ 轴 |
| 缺 $y$ 项（$B = 0$） | 平行于 $y$ 轴 |
| 缺 $z$ 项（$C = 0$） | 平行于 $z$ 轴（垂直于 $xOy$ 面） |
| 缺 $x,y$ 项（$A = B = 0$） | 平行于 $xOy$ 面 |
| $D = 0$ 且缺一项 | 过某条坐标轴 |

### 3. 平面的截距式方程

$$\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$$

其中 $a,b,c$ 分别为平面在 $x,y,z$ 轴上的截距。

### 4. 三点式平面方程

过三点 $M_1, M_2, M_3$ 的平面：

$$\begin{vmatrix}
x - x_1 & y - y_1 & z - z_1 \\
x_2 - x_1 & y_2 - y_1 & z_2 - z_1 \\
x_3 - x_1 & y_3 - y_1 & z_3 - z_1
\end{vmatrix} = 0$$

### 5. 两平面的夹角

平面 $\Pi_1: A_1x + B_1y + C_1z + D_1 = 0$
平面 $\Pi_2: A_2x + B_2y + C_2z + D_2 = 0$

夹角 $\theta$（锐角）：
$$\cos\theta = \frac{|A_1A_2 + B_1B_2 + C_1C_2|}{\sqrt{A_1^2 + B_1^2 + C_1^2} \cdot \sqrt{A_2^2 + B_2^2 + C_2^2}}$$

### 6. 两平面的位置关系

| 关系 | 条件 |
|:---|:---|
| 平行 | $\frac{A_1}{A_2} = \frac{B_1}{B_2} = \frac{C_1}{C_2} \neq \frac{D_1}{D_2}$ |
| 重合 | $\frac{A_1}{A_2} = \frac{B_1}{B_2} = \frac{C_1}{C_2} = \frac{D_1}{D_2}$ |
| 垂直 | $A_1A_2 + B_1B_2 + C_1C_2 = 0$ |

### 7. 点到平面的距离

点 $M_0(x_0, y_0, z_0)$ 到平面 $Ax + By + Cz + D = 0$ 的距离：

$$d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}}$$

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：平面一般式 $Ax + By + Cz + D = 0$ 中，$(A,B,C)$ **就是法向量**，不需要再变换
* 🚨 **易错雷区**：两平面夹角是**锐角**，所以公式中取**绝对值**。如果计算结果为负，取绝对值
* 🚨 **易错雷区**：平行条件中 $\frac{D_1}{D_2}$ 的比值不等——如果全部成比例包括 $D$，那就是重合而不是平行
* 🔑 **秒杀秘籍**：求平面方程的三步法：① 找法向量 ② 找已知点 ③ 代入点法式
* 🔑 **秒杀秘籍**：法向量的求法：已知两条在平面内的向量 → 叉积得法向量；已知三点 → 两个方向向量叉积
* 🔑 **秒杀秘籍**：判断点是否在平面上 → 代入平面方程看等式是否成立

## 三、 闭卷真题挑战

> **【真题演练】**：求过三点 $M_1(1,1,-1)$，$M_2(-2,-2,2)$，$M_3(1,-1,2)$ 的平面方程。

> **点击查看答案与解析**
> **【正确答案】**：
> $\overrightarrow{M_1M_2} = (-3,-3,3)$，$\overrightarrow{M_1M_3} = (0,-2,3)$
>
> 法向量 $\vec{n} = \overrightarrow{M_1M_2} \times \overrightarrow{M_1M_3} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ -3 & -3 & 3 \\ 0 & -2 & 3 \end{vmatrix}$
> $= \vec{i}(-9+6) - \vec{j}(-9-0) + \vec{k}(6-0) = (-3, 9, 6)$
>
> 取 $\vec{n} = (-1, 3, 2)$（同方向可约分），代入点 $M_1(1,1,-1)$：
> $-1(x-1) + 3(y-1) + 2(z+1) = 0$
> $-x + 1 + 3y - 3 + 2z + 2 = 0$
> $-x + 3y + 2z = 0$
>
> **【核心解析】**：
> 求平面方程的标准流程：三点取两点得两个方向向量，叉积得法向量，再代入点法式。法向量可以约去公因子简化。

> **【真题演练】**：求点 $(1,2,1)$ 到平面 $x + 2y + 2z - 10 = 0$ 的距离。

> **点击查看答案与解析**
> **【正确答案】**：
> $$d = \frac{|1 + 2 \times 2 + 2 \times 1 - 10|}{\sqrt{1^2 + 2^2 + 2^2}} = \frac{|1 + 4 + 2 - 10|}{\sqrt{9}} = \frac{|-3|}{3} = 1$$
>
> **【核心解析】**：
> 直接代入点到平面距离公式。分子取绝对值保证距离非负，注意各项符号不要弄错。
