# 8.1 常数项级数的概念与性质

> **学科系统**：高等数学 → 无穷级数 → 常数项级数
> **秒杀类比**：无穷级数就是"一直加下去能不能得到一个有限的数"——像往杯子里倒水，每次倒前一次的一半，永远倒不满但无限接近满杯。级数收敛就是"加无限多次结果有限"，发散就是"加无限多次结果无穷大或没有结果"。

## 一、 核心知识解构

### 1. 级数的定义

$$\sum_{n=1}^\infty a_n = a_1 + a_2 + a_3 + \cdots + a_n + \cdots$$

### 2. 部分和与收敛性

**部分和**：$S_n = a_1 + a_2 + \cdots + a_n = \sum_{k=1}^n a_k$

**收敛**：若 $\lim_{n\to\infty} S_n = S$（有限值），则级数收敛，$S$ 为和。

**发散**：若极限不存在或为无穷大，则级数发散。

### 3. 重要级数的敛散性

| 级数类型 | 形式 | 条件 | 结论 |
|:---|:---|:---:|:---:|
| **等比级数** | $\sum_{n=0}^\infty ar^n$ | $\|r\| < 1$ | 收敛，和 $= \frac{a}{1-r}$ |
| | | $\|r\| \geq 1$ | 发散 |
| **$p$ 级数** | $\sum_{n=1}^\infty \frac{1}{n^p}$ | $p > 1$ | 收敛 |
| | | $p \leq 1$ | 发散 |
| **调和级数** | $\sum_{n=1}^\infty \frac{1}{n}$ | — | 发散（$p=1$ 的 $p$ 级数）|

### 4. 级数的基本性质

| 性质 | 内容 |
|:---|:---|
| **线性** | 若 $\sum a_n = S$，$\sum b_n = T$，则 $\sum (ka_n + lb_n) = kS + lT$ |
| **添加/去掉有限项** | 不改变级数的敛散性 |
| **加括号** | 收敛级数加括号后仍收敛（逆命题不成立） |
| **必要条件** | 若 $\sum a_n$ 收敛，则 $\lim_{n\to\infty} a_n = 0$ |
| **逆否命题** | 若 $\lim_{n\to\infty} a_n \neq 0$，则级数 **发散** |

> **⚠️ 重要警告**：$\lim a_n = 0$ 是收敛的**必要非充分条件**——调和级数 $\sum \frac{1}{n}$ 满足 $a_n \to 0$ 但发散！

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：级数收敛的必要条件是 $\lim a_n = 0$，但反过来不成立——$a_n \to 0$ 不能推出级数收敛
* 🚨 **易错雷区**：加括号后的级数收敛不能推出原级数收敛（反例：$1-1+1-1+\cdots$）
* 🚨 **易错雷区**：不要混淆"数列的极限"和"级数的和"——前者是 $a_n$ 的极限，后者是 $S_n$ 的极限
* 🔑 **秒杀秘籍**：看到 $\sum a_n$，先看 $\lim a_n$ 是否等于 0——如果不等于 0 直接判定发散，不用再做其他审敛法
* 🔑 **秒杀秘籍**：熟记等比级数和 $p$ 级数的敛散性——很多级数的比较审敛法以它们为标准
* 🔑 **秒杀秘籍**：等比级数 $\sum ar^n$ 公比 $|r| < 1$ 时收敛，和 $= \frac{a}{1-r}$

## 三、 闭卷真题挑战

> **【真题演练】**：判断级数 $\sum_{n=1}^\infty \frac{n}{n+1}$ 的敛散性。

> **点击查看答案与解析**
> **【正确答案】**：
> $\lim_{n\to\infty} a_n = \lim_{n\to\infty} \frac{n}{n+1} = 1 \neq 0$
>
> 不满足收敛的必要条件，故级数**发散**。
>
> **【核心解析】**：
> 先看通项是否趋于 0 是最快的判断方法。这里 $a_n \to 1 \neq 0$，直接判定发散。

> **【真题演练】**：求级数 $\sum_{n=1}^\infty \frac{1}{3^{n-1}}$ 的和。

> **点击查看答案与解析**
> **【正确答案】**：
> 该级数是等比级数，$a = 1$，$r = \frac{1}{3}$，$|r| = \frac{1}{3} < 1$
>
> 和 $S = \frac{a}{1-r} = \frac{1}{1-\frac{1}{3}} = \frac{1}{\frac{2}{3}} = \frac{3}{2}$
>
> **【核心解析】**：
> 识别等比级数并判断公比小于 1 → 用求和公式直接算。注意首项是 $n=1$ 时 $a_1 = \frac{1}{3^{0}} = 1$。
