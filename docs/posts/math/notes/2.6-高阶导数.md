# 2.6 高阶导数

> **学科系统**：高等数学 → 一元函数微分学 → 高阶导数
> **秒杀类比**：一阶导数是速度，二阶导数是加速度。每求一次导，就描述更高一层的"变化率的变化率"。

## 一、 核心知识解构

### 1. 高阶导数定义

$f(x)$ 的 $n$ 阶导数是 $n-1$ 阶导数的导数。

记法：$f^{(n)}(x)$ 或 $\frac{d^n y}{dx^n}$

### 2. 常用高阶导数公式

| 原函数 | $n$ 阶导数 |
|:---|:---|
| $x^m$ | $m(m-1)\cdots(m-n+1)x^{m-n}$ |
| $e^{ax}$ | $a^n e^{ax}$ |
| $\sin(ax+b)$ | $a^n \sin(ax+b+\frac{n\pi}{2})$ |
| $\cos(ax+b)$ | $a^n \cos(ax+b+\frac{n\pi}{2})$ |
| $\ln(1+x)$ | $(-1)^{n-1}\frac{(n-1)!}{(1+x)^n}$ |
| $(ax+b)^k$ | $k(k-1)\cdots(k-n+1)a^n(ax+b)^{k-n}$ |

### 3. 莱布尼兹公式（乘积的高阶导数）

$$ (uv)^{(n)} = \sum_{k=0}^{n} C_n^k u^{(n-k)} v^{(k)} $$

## 二、 考试红牌警告与秒杀秘籍

* 🚨 **易错雷区**：$\sin x$ 的 $n$ 阶导数是 $\sin(x + \frac{n\pi}{2})$，这是**递推规律**——每求一次导就向前平移 $\frac{\pi}{2}$，不是直接乘 $n$！
* 🔑 **秒杀秘籍**：求分式的高阶导数时，先裂项拆成简单分式之和，再用 $\left(\frac{1}{ax+b}\right)^{(n)} = \frac{(-1)^n n! a^n}{(ax+b)^{n+1}}$ 求解。

## 三、 闭卷真题挑战

> **【真题演练】**：求函数 $f(x) = \sin 2x$ 的 $10$ 阶导数。

> **点击查看答案与解析**
> **【正确答案】**：$f^{(10)}(x) = -2^{10} \sin 2x$
>
> **【核心解析】**：
> 利用公式 $\sin(ax+b)$ 的高阶导数：
> $$ f^{(n)}(x) = a^n \sin(ax+b + \frac{n\pi}{2}) = 2^{10} \sin(2x + \frac{10\pi}{2}) = 2^{10} \sin(2x + 5\pi) = -2^{10} \sin 2x $$
