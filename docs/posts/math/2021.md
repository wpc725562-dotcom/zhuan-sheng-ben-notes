# 高等数学 · 2021 年真题演练

> 类型：同型演练（B/C 级资料）| 主攻：**微分方程与变限积分**
> 边界：完整原卷以正版真题册为准


## 计算题

**1.** 解 \(y'+y=e^{-x}\)。

**答案：** \(y=e^{-x}(x+C)\)
**讲解：** 一阶线性，积分因子 \(e^{\int Pdx}=e^x\)。

**2.** 设 \(F(x)=\displaystyle\int_0^{x^2}\sin t\,dt\)，求 \(F'(x)\)。

**答案：** \(F'(x)=2x\sin(x^2)\)
**讲解：** 变限 + 链式法则。

**3.** 求 \(\displaystyle\int\frac{2x+1}{x^2+x}\,dx\)。

**答案：** \(\ln|x^2+x|+C\)

## 得分点
- 微分方程写通解含任意常数
- 变限积分先对上限求导再乘内层导数
