# 高等数学 · 2020 年真题演练

> 类型：同型演练（B/C 级资料）| 主攻：**积分基础**
> 边界：完整原卷以正版真题册为准。见 资料来源说明

## 题目

### 1. 【选择】

$\int \frac{1}{x}\,dx=$

**答案：** $\ln|x|+C$
**讲解：** 注意绝对值

### 2. 【计算】

$\int x e^{x}\,dx$

**答案：** $e^x(x-1)+C$
**讲解：** 分部积分 u=x

### 3. 【计算】

$\int_0^1 (2x+1)\,dx$

**答案：** 2
**讲解：** 牛顿-莱布尼茨公式

### 4. 【计算】

$\int \frac{1}{1+x^2}\,dx$

**答案：** $\arctan x+C$
**讲解：** 基本积分表

### 5. 【应用】

求 y=x^2 与 x=1、y=0 围成图形绕 x 轴旋转体积

**答案：** $\pi/5$
**讲解：** V=π∫₀¹ x⁴ dx

## 得分策略

- 极限：等价无穷小优先，再洛必达
- 导数：复合 / 隐函数 / 参数方程分型训练
- 积分：先看是否凑微分，再分部
