## 学习构造（建议路径）

1. **先做单选 + 填空**（送分区，保证正确率）
2. **再刷计算题三大件**：极限 / 求导 / 积分
3. **大题套路**：微分方程特征根、二重积分换序/极坐标、积分方程求导消积分号
4. 对照文末「考点分布」回炉薄弱章
