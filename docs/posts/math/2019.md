# 高等数学 · 2019 年真题演练

> 类型：同型演练（B/C 级资料）| 主攻：**一元积分**
> 边界：完整原卷以正版真题册为准


## 一、填空/选择

**1.** \(\displaystyle\int \frac{1}{1+x^2}\,dx=\) ______ +C

**答案：** \(\arctan x\)

**2.** \(\displaystyle\int_0^1 x\,dx=\) ______

**答案：** \(\dfrac{1}{2}\)

## 二、计算题

**3.** 求 \(\displaystyle\int x e^{x}\,dx\)。

**答案：** \(e^{x}(x-1)+C\)
**讲解：** 分部积分 \(u=x,\,dv=e^x dx\)。

**4.** 求 \(\displaystyle\int_0^{\pi/2}\sin^2 x\,dx\)。

**答案：** \(\dfrac{\pi}{4}\)
**讲解：** \(\sin^2 x=\frac{1-\cos 2x}{2}\)。

**5.** 用换元求 \(\displaystyle\int\frac{1}{\sqrt{a^2-x^2}}\,dx\ (a>0)\)。

**答案：** \(\arcsin\frac{x}{a}+C\)

## 三、得分点
- 不定积分别忘 +C
- 定积分检查上下限代换
- 凑微分先写对 \(du\)
