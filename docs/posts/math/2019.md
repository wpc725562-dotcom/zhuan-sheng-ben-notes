# 高等数学 · 2019 年真题演练

> 类型：同型演练（B/C 级资料）| 主攻：**导数与微分**
> 边界：完整原卷以正版真题册为准。见 资料来源说明

## 题目

### 1. 【选择】

$y=e^{x^2}$ 的导数

**答案：** $2x e^{x^2}$
**讲解：** 链式法则

### 2. 【选择】

可导一定连续？

**答案：** 是
**讲解：** 连续是可导的必要条件，非充分

### 3. 【计算】

求 $y=x^x$（x 大于 0）的导数

**答案：** $x^x(\ln x+1)$
**讲解：** 写成 e^{x ln x} 再求导

### 4. 【计算】

隐函数 $x^2+y^2=1$ 求 $y'$

**答案：** $y'=-x/y$（y≠0）
**讲解：** 两边对 x 求导

### 5. 【应用】

求曲线 $y=x^3-3x$ 的极值点

**答案：** x=±1
**讲解：** y'=3x^2-3=0

## 得分策略

- 极限：等价无穷小优先，再洛必达
- 导数：复合 / 隐函数 / 参数方程分型训练
- 积分：先看是否凑微分，再分部
