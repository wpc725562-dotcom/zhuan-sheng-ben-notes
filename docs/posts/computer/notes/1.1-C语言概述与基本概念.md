# C语言概述与基本概念

> 🎯 **一句话秒杀**：C语言是面向过程的编程语言，程序由**函数**组成，main函数是程序入口。

---

## 一、C语言发展史

| 年份 | 事件 |
|:---|:---|
| 1972 | 丹尼斯·里奇在贝尔实验室设计C语言 |
| 1978 |  Kernighan & Ritchie 出版《C程序设计语言》→ K&R C |
| 1989 | ANSI 发布 C89/C90 标准（最经典） |
| 1999 | C99 标准（新增//注释、变长数组等） |
| 2011 | C11 标准 |
| 2024 | C23 最新标准 |

## 二、第一个C程序

```c
#include <stdio.h>   // 预处理指令——包含标准输入输出头文件

int main()           // 主函数——程序入口
{
    printf("Hello, World!\n");  // 调用库函数输出
    return 0;        // 返回整型0给操作系统，表示正常结束
}
```

### 🚨 红牌警告
- `main` 是程序的**唯一入口**，一个程序有且只有一个 main 函数
- `return 0;` 必不可少——表示程序正常退出（很多考试问这个）
- `#include` 以 `#` 开头——**不需要分号结尾**
- `stdio.h` 是**头文件**，`printf` 和 `scanf` 都在里面

## 三、C程序的执行过程

```
源代码 (.c) → 预处理 → 编译 → 汇编 → 链接 → 可执行文件 (.exe)
```

| 阶段 | 产物 | 说明 |
|:---|:---|:---|
| 预处理 | `.i` 文件 | 处理 `#include`、`#define` 等指令 |
| 编译 | `.s` 文件 | 将C代码转为汇编代码 |
| 汇编 | `.o` / `.obj` 文件 | 将汇编代码转为机器指令 |
| 链接 | `.exe` 文件 | 将多个目标文件+库文件合并成可执行文件 |

### 🔑 秒杀秘籍
> 考试常考：**链接**是把所有目标文件连起来生成最终exe；**编译**只是把单个.c变成.obj

## 四、C语言的基本构成——函数

```c
返回值类型 函数名(参数列表)
{
    语句序列;
    return 返回值;
}
```

**C程序 = 一个 main 函数 + 若干其他函数**

- 函数是C程序的基本单位
- 不允许函数嵌套定义（一个函数内不能定义另一个函数）
- 但可以嵌套调用（函数A里调用函数B，函数B里调用函数C）

## 五、标识符与关键字

### 标识符命名规则
1. 由 **字母、数字、下划线** 组成
2. **不能以数字开头**
3. **区分大小写**（Sum ≠ sum）
4. 不能与关键字重名

✅ 合法：`sum`, `_count`, `student_name`, `MaxValue`
❌ 非法：`2num`（数字开头）, `a+b`（特殊符号）, `int`（关键字）

### 🚨 红牌警告
> 考试最爱考：`2a` ❌ `a_1` ✅ `_int` ✅ `int` ❌（关键字）
> 注意：**下划线可以开头**，`_a` 是合法标识符

### 常见关键字（32个，C89标准）

| 类别   | 关键字                                                                                         |
| :--- | :------------------------------------------------------------------------------------------ |
| 数据类型 | `int` `char` `float` `double` `void` `short` `long` `signed` `unsigned`                     |
| 控制语句 | `if` `else` `switch` `case` `default` `for` `while` `do` `break` `continue` `goto` `return` |
| 构造类型 | `struct` `union` `enum` `typedef`                                                           |
| 存储类别 | `auto` `static` `extern` `register` `const` `volatile`                                      |
| 其他   | `sizeof`                                                                                    |

## 六、预处理指令

| 指令 | 用途 | 示例 |
|:---|:---|:---|
| `#include` | 包含头文件 | `#include <stdio.h>` |
| `#define` | 定义宏常量/宏函数 | `#define PI 3.14159` |
| `#ifdef` / `#endif` | 条件编译 | |

### 🔑 秒杀秘籍
> `#include <文件名>` 在系统头文件目录下搜索（直接查找）
> `#include "文件名"` 先在当前目录搜索，再到系统目录搜索（先局部后全局）

## 📝 闭卷真题挑战

**（点击下方空白查看答案）**

```
1. C程序的基本单位是？
2. main函数的位置？
3. 以下哪个是合法标识符？
   A. 2ab   B. _abc   C. int   D. a+b
4. C程序的执行过程顺序？
5. #include < > 和 #include " " 的区别？
```

<details>
<summary>👆 点击展开答案</summary>

1. **函数**
2. main函数可以出现在程序的**任何位置**——但一个程序有且只有一个main函数
3. **B. _abc**（A数字开头，C关键字，D含特殊符号）
4. **预处理 → 编译 → 汇编 → 链接**
5. `< >` 只在系统目录搜索，`" "` 先当前目录再系统目录

</details>
