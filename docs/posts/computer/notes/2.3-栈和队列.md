# 栈和队列

> 🎯 **一句话秒杀**：**栈**是"先进后出"（后放的先拿），**队列**是"先进先出"（先排的先走）。都是操作受限的线性表。

---

## 一、栈（Stack）— 后进先出 LIFO

### 栈的定义

```
栈顶 ← ← 出栈/入栈都在这里
  ┌─────┐
  │ an  │ ← 栈顶
  │ ... │
  │ a2  │
  │ a1  │ ← 栈底
  └─────┘
```

**核心原则：** 只能在**栈顶**操作——进栈（push）和出栈（pop）

### 顺序栈（数组实现）

```c
#define MAXSIZE 100

typedef struct {
    int data[MAXSIZE];  // 存放元素
    int top;             // 栈顶指针
} SqStack;

// 初始化
void InitStack(SqStack *S) {
    S->top = -1;  // 空栈时top=-1
}

// 判空
int IsEmpty(SqStack S) {
    return S.top == -1;
}

// 入栈
int Push(SqStack *S, int x) {
    if (S->top == MAXSIZE-1) return 0;  // 栈满
    S->data[++S->top] = x;  // top先+1，再赋值
    return 1;
}

// 出栈
int Pop(SqStack *S, int *x) {
    if (IsEmpty(*S)) return 0;  // 栈空
    *x = S->data[S->top--];     // 取值，top再-1
    return 1;
}

// 读栈顶
int GetTop(SqStack S, int *x) {
    if (IsEmpty(S)) return 0;
    *x = S.data[S.top];
    return 1;
}
```

### 链栈（链表实现）

```c
// 用单链表实现——入栈就是头插，出栈就是删第一个结点
typedef struct Node {
    int data;
    struct Node *next;
} *LinkStack;

// 入栈——头插
void Push(LinkStack *S, int x) {
    Node *p = (Node *)malloc(sizeof(Node));
    p->data = x;
    p->next = *S;
    *S = p;
}

// 出栈——删除头结点
int Pop(LinkStack *S, int *x) {
    if (*S == NULL) return 0;
    *x = (*S)->data;
    Node *p = *S;
    *S = (*S)->next;
    free(p);
    return 1;
}
```

### 🚨 红牌警告
> **top = -1 和 top = 0 是两种不同的实现风格！**
> - `top = -1`：先加再赋值，`S.data[++top] = x`
> - `top = 0`：先赋值再加，`S.data[top++] = x`
> 考试注意看清楚题目用的哪种！

## 二、栈的应用

### 1. 括号匹配

```c
// 思想：遇到左括号入栈，遇到右括号出栈匹配
int Match(char exp[]) {
    SqStack S;
    InitStack(&S);
    for (int i = 0; exp[i]; i++) {
        if (exp[i] == '(' || exp[i] == '[')
            Push(&S, exp[i]);
        else if (exp[i] == ')') {
            char top; Pop(&S, &top);
            if (top != '(') return 0;
        } else if (exp[i] == ']') {
            char top; Pop(&S, &top);
            if (top != '[') return 0;
        }
    }
    return IsEmpty(S);  // 全部匹配完还要确保栈空！
}
```

### 2. 表达式求值

```
中缀表达式：  (3+4)×5-6
后缀表达式：  34+5×6-     ← 用栈求值
```

**后缀表达式求值过程：**
1. 遇到数字 → 入栈
2. 遇到运算符 → 弹出两个数计算 → 结果入栈
3. 最后栈顶就是结果

### 3. 函数递归调用

每次函数调用都会在系统栈中压入**返回地址和局部变量**
递归太深会**栈溢出**（Stack Overflow）

## 三、队列（Queue）— 先进先出 FIFO

### 队列的定义

```
出队 ← ┌──┬──┬──┬──┬──┐ ← 入队
       │a1│a2│...│an│  │
         ↑              ↑
       队头            队尾
```

**核心原则：** 队头出、队尾入

### 循环队列

为什么用循环？为了避免"假溢出"（队尾到顶了但队头还有空位）

```c
#define MAXSIZE 100

typedef struct {
    int data[MAXSIZE];
    int front;  // 队头指针（指向队头元素）
    int rear;   // 队尾指针（指向队尾元素的下一位置）
} SqQueue;

// 初始化
void InitQueue(SqQueue *Q) {
    Q->front = Q->rear = 0;  // front=rear=0为空
}

// 判空
int IsEmpty(SqQueue Q) {
    return Q.front == Q.rear;
}

// 判满（牺牲一个单元区分空/满）
int IsFull(SqQueue Q) {
    return (Q.rear + 1) % MAXSIZE == Q.front;
}

// 入队
int EnQueue(SqQueue *Q, int x) {
    if (IsFull(*Q)) return 0;
    Q->data[Q->rear] = x;
    Q->rear = (Q->rear + 1) % MAXSIZE;  // 循环移动
    return 1;
}

// 出队
int DeQueue(SqQueue *Q, int *x) {
    if (IsEmpty(*Q)) return 0;
    *x = Q->data[Q->front];
    Q->front = (Q->front + 1) % MAXSIZE;
    return 1;
}
```

### 🔑 秒杀秘籍

> **循环队列核心公式（必背！）：**
> - 入队：`rear = (rear + 1) % MAXSIZE`
> - 出队：`front = (front + 1) % MAXSIZE`
> - 判空：`front == rear`
> - **判满：`(rear + 1) % MAXSIZE == front`**（牺牲一个元素空间）
> - 求长度：`(rear - front + MAXSIZE) % MAXSIZE`

### 链队列

```c
// 用带头结点的单链表实现
typedef struct Node {
    int data;
    struct Node *next;
} QNode;

typedef struct {
    QNode *front;  // 队头（删除端）
    QNode *rear;   // 队尾（插入端）
} LinkQueue;
```

## 四、栈 vs 队列

| 对比 | 栈 | 队列 |
|:---|:---|:---|
| 原则 | 后进先出 (LIFO) | 先进先出 (FIFO) |
| 操作端 | 只在**栈顶**操作 | 队头出、队尾入 |
| 应用 | 括号匹配、递归、表达式求值 | 层次遍历、缓冲区、BFS |
| 类比 | 一摞盘子 | 排队买票 |
