# 计算机程序设计 · 2018 年真题演练

> 主攻：**C基础与线性表** | 同型演练


## 选择/判断
**1.** C 语言中，下列标识符合法的是（ ）
A. 2a  B. `_sum`  C. int  D. a-b
**答案：B**

**2.** 顺序表的特点是（随机存取；存储密度高）。链表的特点是（插入删除方便；需指针域）。

## 简答
**顺序表与单链表的优缺点对比。**
要点：存取方式、插入删除时间复杂度、空间利用、是否需要连续空间。

## 读程序
```c
#include <stdio.h>
int main(){
    int a=3,b=4,c;
    c=a>b?a:b;
    printf("%d",c);
    return 0;
}
```
**输出：** 4
