## 同型题

**1.** switch 中 case 后通常要

**答案：** break
**考点：** 1.4

**2.** for(;;) 是

**答案：** 无限循环
**考点：** 1.5

**3.** 队列特点

**答案：** 先进先出 FIFO
**考点：** 2.3

**4.** 循环队列牺牲一个单元是为了

**答案：** 区分队空队满
**考点：** 2.3

**5.** 手写：求 1 到 n 求和函数

**答案：** int sum(int n){int s=0;for(int i=1;i<=n;i++)s+=i;return s;}
**考点：** 1.7
