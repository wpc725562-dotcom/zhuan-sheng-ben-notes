# 📐 语法考点精讲

> 专升本英语核心语法 | 时态/语态/从句/虚拟语气/非谓语

---

## 一、动词时态（🔥 必考）

### ⏰ 三大核心时态

| 时态 | 结构 | 用法 | 标志词 |
|:---|:---|:---|:---|
| **一般现在时** | do/does | 习惯、真理、状态 | always, usually, often, every... |
| **一般过去时** | did | 过去发生的动作 | yesterday, last..., ago, in 2020 |
| **一般将来时** | will do | 将要发生的动作 | tomorrow, next..., soon, in the future |

### 🔥 完形/语法填空最高频时态

| 时态 | 结构 | 用法 |
|:---|:---|:---|
| **现在完成时** | have/has done | 过去发生对现在有影响 / 持续到现在 |
| **过去完成时** | had done | 过去的过去（站在过去看更早） |
| **现在进行时** | am/is/are doing | 此刻正在发生 |
| **过去进行时** | was/were doing | 过去某时刻正在发生 |
| **现在完成进行时** | have/has been doing | 从过去到现在一直持续 |

> **真题规律：** 80%的时态题在 **一般过去时**、**现在完成时**、**一般现在时** 三者之间选择

### 📌 现在完成时 vs 一般过去时（必考辨析）

| 维度 | 现在完成时 | 一般过去时 |
|:---|:---|:---|
| 时间焦点 | **现在**（关注结果） | **过去**（关注动作本身） |
| 能否用具体时间 | ❌ 不能 | ✅ 可以 |
| 例句 | I have seen the movie.（现在知道内容） | I saw the movie last night. |

**标志词区分：**
- 完成时：already, yet, ever, never, just, recently, so far, since, for
- 过去时：yesterday, last..., ...ago, in 2020, when I was...

## 二、被动语态

### 结构公式

```
be + done (过去分词)
```

### 各时态被动语态

| 时态 | 被动结构 |
|:---|:---|
| 一般现在时 | am/is/are + done |
| 一般过去时 | was/were + done |
| 一般将来时 | will be + done |
| 现在完成时 | have/has been + done |
| 过去完成时 | had been + done |
| 情态动词 | can/must/may + be + done |

### 易错点

- **主动表被动：** need/want/require + doing = need to be done
  - *The car needs **washing**.* = The car needs **to be washed**.
- **感官动词：** look, sound, feel, taste, smell + 形容词（主动表被动）
  - *The flower **smells good**.*

## 三、三大从句（🔥🔥 必考）

### 1️⃣ 定语从句

| 先行词 | 人 | 物 | 人+物/所有格 |
|:---|:---|:---|:---|
| 主格 | who/that | which/that | that |
| 宾格 | who(m)/that | which/that | that |
| 所有格 | whose | whose/of which | whose |

**非限制性定语从句：** 不能用 that，用 which / who

> *The book, **which** I bought yesterday, is very interesting.*

### 2️⃣ 名词性从句

| 类型 | 引导词 | 例句 |
|:---|:---|:---|
| 主语从句 | that / whether / what | **What** he said is true. |
| 宾语从句 | that / if/whether / wh- | I think **that** he is right. |
| 表语从句 | that / whether / wh- | The question is **whether** we should go. |
| 同位语从句 | that | The news **that** he won surprised us. |

### 3️⃣ 状语从句

| 类型 | 引导词 |
|:---|:---|
| 时间 | when, while, as, before, after, until, as soon as |
| 条件 | if, unless, as long as, provided that |
| 原因 | because, since, as, now that |
| 让步 | although, though, even though, while, no matter... |
| 目的 | so that, in order that |
| 结果 | so...that, such...that |

## 四、虚拟语气（🔥 必考）

### 三类条件句对比

| 类型 | if从句 | 主句 |
|:---|:---|:---|
| **与现在相反** | **did/were** | **would/could/might + do** |
| **与过去相反** | **had done** | **would/could/might + have done** |
| **与将来相反** | **should do / were to do** | **would/could/might + do** |

> **专升本规律：** 80%的虚拟语气题只考前两种（与现在/过去相反）

### 其他虚拟语气考点

| 考点 | 结构 | 例句 |
|:---|:---|:---|
| 建议/命令/要求 | **should do**（should可省） | He suggested that she (should) go. |
| wish后 | 时态退一格 | I wish I **were** a bird. |
| as if/though | 时态退一格 | He talks as if he **knew** everything. |
| It's time that... | 用过去式 | It's time that we **left**. |
| would rather that... | 用过去式 | I'd rather you **came** tomorrow. |

**动词清单（接 that 虚拟）：** suggest, advise, propose, recommend, insist, order, command, require, request, demand

## 五、非谓语动词（🔥🔥 必考）

| 形式 | 意义 | 用法 |
|:---|:---|:---|
| **to do** | 目的、将来 | 不定式：表示目的/主动/将来 |
| **doing** | 主动、进行 | 现在分词/动名词 |
| **done** | 被动、完成 | 过去分词表示被动/完成 |

### 常考固定搭配

**只接 to do 的动词：**
> want, hope, wish, expect, decide, plan, refuse, promise, afford, agree

**只接 doing 的动词：**
> enjoy, finish, avoid, consider, practice, suggest, mind, keep, imagine

**既接 to do 又接 doing 意义不同：**

| 动词 | to do | doing |
|:---|:---|:---|
| **remember** | 记得要去做 | 记得做过 |
| **forget** | 忘记要去做 | 忘记做过 |
| **stop** | 停下来去做 | 停止做 |
| **try** | 努力去做 | 尝试做 |
| **regret** | 遗憾要去做 | 后悔做过 |

## 六、语法填空解题技巧

### 纯填空（无提示词）

| 空格位置 | 可能填的词 |
|:---|:---|
| 名词前 | 冠词(a/an/the)、限定词、介词 |
| 动词前 | 情态动词、助动词 |
| 两句之间 | 连词(and, but, or, when, because...) |
| 句中 | 关系词(who, which, that, where...) |

### 有提示词

| 提示词词性 | 可能变形成 |
|:---|:---|
| 动词 | 时态、语态、非谓语（to do/doing/done） |
| 形容词 | 副词(-ly)、比较级/最高级 |
| 名词 | 复数、词性转换（→形容词） |
