## 复盘法（本年通用）

1. **阅读**：题干关键词 → 回原文定位 → 排除绝对化选项
2. **完形**：先通读 → 搭配优先 → 语法兜底
3. **词汇语法**：时态语态 / 非谓语 / 从句 / 固定搭配
4. **写作**：每周默写 1 篇同体裁
5. **错题登记**：单词 | 错因 | 正确搭配
